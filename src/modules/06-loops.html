Módulo 06
Loops
Loops executam um bloco de código repetidamente enquanto uma condição for verdadeira ou para cada item de uma coleção.
Loop for
Dart
void main() {
  for (int i = 1; i <= 5; i++) {
    print('Contagem: $i');
  }
}
O for tem três partes: inicialização (int i = 1) · condição (i <= 5) · incremento (i++). Executa enquanto a condição for true.
for-in
Use for-in para percorrer todos os itens de uma lista sem precisar de índice.
Dart
void main() {
  var frutas = ['maçã', 'banana', 'uva'];

  for (var fruta in frutas) {
    print(fruta);
  }
}
while
O while repete o bloco enquanto a condição for true. Use quando não sabe quantas repetições serão necessárias.
Dart
void main() {
  int n = 1;

  while (n <= 5) {
    print('n = $n');
    n++;
  }
}
break e continue
Dart
void main() {
  for (int i = 1; i <= 10; i++) {
    if (i == 4) continue; // pula o 4
    if (i == 7) break;    // para no 7
    print(i);
  }
  // exibe: 1 2 3 5 6
}
Exercícios de fixação
01
Tabuada
Loop for com cálculo
Use um for para exibir a tabuada do 7 (de 7×1 até 7×10), no formato "7 x 1 = 7".
Solução
void main() {
  for (int i = 1; i <= 10; i++) {
    print('7 x $i = ${7 * i}');
  }
}
02
Só os ímpares
for com continue
Use um for de 1 a 20 e exiba apenas os números ímpares usando continue para pular os pares.
Solução
void main() {
  for (int i = 1; i <= 20; i++) {
    if (i % 2 == 0) continue;
    print(i);
  }
}
