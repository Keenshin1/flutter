Módulo 05
if / else
Condicionais permitem que o programa tome decisões — executando caminhos diferentes dependendo se uma condição é verdadeira ou falsa.
Operadores de comparação
Para criar condições precisamos comparar valores. Toda comparação retorna um bool (true ou false).
Dart
void main() {
  print(5 == 5);  // true   igual
  print(5 != 3);  // true   diferente
  print(5 >  3);  // true   maior que
  print(5 <  3);  // false  menor que
  print(5 >= 5);  // true   maior ou igual
  print(5 <= 4);  // false  menor ou igual
}
Você estudou esses operadores com detalhes no módulo anterior. Aqui os usaremos para criar condições no código.
if / else
O if executa um bloco de código somente se a condição for true. O else é executado quando a condição é false.
Dart
void main() {
  int idade = 20;

  if (idade >= 18) {
    print('Maior de idade');
  } else {
    print('Menor de idade');
  }
}
else if — múltiplas condições
Quando há mais de dois casos possíveis, encadeie condições com else if. O Dart avalia de cima para baixo e executa apenas o primeiro bloco cuja condição for verdadeira.
Dart
void main() {
  double nota = 7.5;

  if (nota >= 9) {
    print('Excelente');
  } else if (nota >= 7) {
    print('Aprovado');
  } else if (nota >= 5) {
    print('Recuperação');
  } else {
    print('Reprovado');
  }
}
A ordem importa. Sempre verifique as condições do caso mais específico para o mais geral. No exemplo acima, se invertêssemos a ordem (verificar ≥5 antes de ≥7), a nota 7.5 seria classificada como "Recuperação" por engano.
Exercícios de fixação
01
Maior ou menor de idade
if / else simples
Básico
Dada uma variável int idade = 16, use if/else para exibir "Pode votar" se a idade for maior ou igual a 16, ou "Não pode votar" caso contrário.
Solução
void main() {
  int idade = 16;

  if (idade >= 16) {
    print('Pode votar');
  } else {
    print('Não pode votar');
  }
}
02
Classificador de notas
else if encadeado
Básico
Dada uma variável double nota = 7.5, use if/else if/else para classificar: ≥9 → "A", ≥7 → "B", ≥5 → "C", abaixo → "D".
Solução
void main() {
  double nota = 7.5;

  if (nota >= 9) {
    print('A');
  } else if (nota >= 7) {
    print('B');
  } else if (nota >= 5) {
    print('C');
  } else {
    print('D');
  }
}
03
Classificador de velocidade
Múltiplas condições encadeadas
Desafio
Dada int velocidade = 95, classifique: abaixo de 40 → "Devagar", entre 40 e 59 → "Moderado", entre 60 e 79 → "Normal", entre 80 e 109 → "Rápido", 110 ou mais → "Muito rápido". Em seguida, use um if/else separado para exibir "Multa!" se a velocidade ultrapassar 80.
Solução
void main() {
  int velocidade = 95;

  if (velocidade < 40) {
    print('Devagar');
  } else if (velocidade < 60) {
    print('Moderado');
  } else if (velocidade < 80) {
    print('Normal');
  } else if (velocidade < 110) {
    print('Rápido');
  } else {
    print('Muito rápido');
  }

  if (velocidade > 80) {
    print('Multa!');
  }
}
