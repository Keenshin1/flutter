Módulo 10
Mapas
Um mapa armazena pares de chave → valor. Ideal para representar dados estruturados como configurações, dicionários e registros.
Criando e acessando mapas
Dart
void main() {
  Map<String, int> idades = {
    'Tainá': 28,
    'Bruno': 28,
    'Carla': 28,
  };

  print(idades['Tainá']);  // 28
  print(idades.length);   // 3
  print(idades.keys);     // (Tainá, Bruno, Carla)
  print(idades.values);   // (28, 28, 28)
}
Modificando mapas
Dart
void main() {
  var cores = {'céu': 'azul', 'grama': 'verde'};

  cores['sol'] = 'amarelo';  // adiciona nova entrada
  cores['céu'] = 'cinza';   // atualiza valor existente
  cores.remove('grama');    // remove pela chave

  print(cores.containsKey('sol'));   // true
  print(cores.containsKey('grama')); // false
  print(cores);
}
Acesso por chave retorna um valor nullable
Quando você acessa um mapa por chave (mapa[chave]), o Dart retorna o tipo nullable (T?) — porque a chave pode não existir e o resultado seria null. Use o operador ?? para fornecer um valor padrão seguro.
Dart
void main() {
  var notas = {'Tainá': 9.0, 'Bruno': 7.5};

  // mapa[chave] retorna double? (pode ser null)
  double? n1 = notas['Tainá'];   // 9.0
  double? n2 = notas['Diego'];   // null (chave inexistente)

  print(n1); // 9.0
  print(n2); // null

  // Use ?? para garantir um valor padrão
  double notaDiego = notas['Diego'] ?? 0.0;
  print(notaDiego); // 0.0

  // containsKey evita acessar chaves inexistentes
  if (notas.containsKey('Bruno')) {
    print(notas['Bruno']); // 7.5
  }
}
Nunca use mapa[chave]! (com !) sem verificar antes se a chave existe — se não existir, o programa lança um erro em tempo de execução. Prefira ?? valorPadrão ou containsKey.
Percorrendo um mapa
Dart
void main() {
  var notas = {'Tainá': 9.0, 'Bruno': 7.5, 'Carla': 8.2};

  // forEach com chave e valor
  notas.forEach((nome, nota) {
    print('$nome: $nota');
  });

  // for-in com entries
  for (var entry in notas.entries) {
    print('${entry.key} tirou ${entry.value}');
  }
}
Exercícios de fixação
01
Agenda telefônica
Criando e consultando
Básico
Crie um Map<String, String> com nomes e telefones de 3 contatos. Adicione um 4º contato, remova um existente e exiba todos os nomes e o tamanho final.
Solução
void main() {
  Map<String, String> agenda = {
    'Tainá': '9999-1111',
    'Bruno': '9999-2222',
    'Carla': '9999-3333',
  };
  agenda['Diego'] = '9999-4444';
  agenda.remove('Bruno');
  print(agenda.keys);
  print(agenda.length);
}
02
Relatório de notas
Percorrendo com forEach
Básico
Dado o mapa {'Tainá': 8.5, 'Bob': 6.0, 'Clara': 9.5}, use forEach para exibir cada aluno no formato: "Tainá: 8.5 - Aprovado" (aprovado se nota ≥ 7).
Solução
void main() {
  var notas = {'Tainá': 8.5, 'Bob': 6.0, 'Clara': 9.5};
  notas.forEach((nome, nota) {
    var status = nota >= 7 ? 'Aprovado' : 'Reprovado';
    print('$nome: $nota - $status');
  });
}
03
Inventário de loja
Map com operações avançadas
Desafio
Crie um Map<String, int> representando um estoque (produto → quantidade) com pelo menos 5 itens. Percorra com forEach e exiba apenas os produtos com estoque abaixo de 5 como "Estoque crítico". Por fim, use um for-in em .values para calcular e exibir o total de itens em estoque.
Solução
void main() {
  var estoque = {
    'Café':      3,
    'Leite':     12,
    'Pão':       2,
    'Manteiga':  8,
    'Ovos':      1,
  };

  estoque.forEach((produto, qtd) {
    if (qtd < 5) print('$produto: Estoque crítico ($qtd)');
  });

  int total = 0;
  for (var qtd in estoque.values) total += qtd;
  print('Total em estoque: $total');
}

Exercício integrador
Combine funções, listas e mapas num único programa.
R
Gerenciador de alunos
Funções · Listas · Mapas
Revisão
Crie uma função String status(double nota) que retorne "Aprovado" (≥7), "Recuperação" (≥5) ou "Reprovado". Dado o mapa {'Tainá': 8.5, 'Bob': 4.0, 'Clara': 6.5, 'Diego': 9.0}, percorra-o e exiba o nome + o resultado de status(nota) para cada aluno. Por fim, use where() nas chaves para criar e exibir uma lista apenas com os aprovados.
Solução
String status(double nota) {
  if (nota >= 7) return 'Aprovado';
  if (nota >= 5) return 'Recuperação';
  return 'Reprovado';
}

void main() {
  var turma = {'Tainá': 8.5, 'Bob': 4.0, 'Clara': 6.5, 'Diego': 9.0};

  turma.forEach((nome, nota) {
    print('$nome: ${status(nota)}');
  });

  var aprovados = turma.keys
      .where((nome) => (turma[nome] ?? 0) >= 7)
      .toList();
  print('Aprovados: $aprovados');
}
