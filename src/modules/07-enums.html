Módulo 07
enum
Enums criam um conjunto fixo de valores nomeados — perfeitos para representar estados, categorias e opções predefinidas.
O que é um enum?
Imagine que você quer guardar o dia da semana. Poderia usar um número (1 a 7) ou uma String ("segunda"), mas ambos permitem valores inválidos como 9 ou "banana". Um enum restringe as opções exatamente ao que faz sentido.
Dart
enum DiaDaSemana {
  segunda,
  terca,
  quarta,
  quinta,
  sexta,
  sabado,
  domingo,
}

void main() {
  DiaDaSemana hoje = DiaDaSemana.sexta;
  print(hoje);           // DiaDaSemana.sexta
  print(hoje.name);      // sexta
}
Por convenção, o nome do enum usa PascalCase (DiaDaSemana) e os valores usam camelCase (segunda, terca). A propriedade .name retorna o nome do valor como String.
Enum com switch
Enums e switch funcionam muito bem juntos. Você verá isso constantemente no Flutter — por exemplo, ao verificar o estado de uma conexão ou o tipo de um botão.
Dart
enum Estacao { primavera, verao, outono, inverno }

void main() {
  Estacao atual = Estacao.inverno;

  switch (atual) {
    case Estacao.primavera:
      print('Flores brotando!');
    case Estacao.verao:
      print('Calor intenso.');
    case Estacao.outono:
      print('Folhas caindo.');
    case Estacao.inverno:
      print('Frio e casaco!');
  }
}
Saída: Frio e casaco!
No Dart 3+ o switch é exaustivo para enums: o compilador avisa se você esquecer de cobrir algum valor. Isso evita bugs silenciosos — uma das grandes vantagens de usar enums em vez de strings.
Enum com propriedades (enhanced enum)
Desde o Dart 2.17, enums podem ter propriedades e métodos. Isso é poderoso: ao invés de só nomear valores, você pode associar dados a cada um.
Dart
enum Planeta {
  mercurio(0.39),
  venus(0.72),
  terra(1.00),
  marte(1.52);

  const Planeta(this.distanciaUA);
  final double distanciaUA;
}

void main() {
  print(Planeta.marte.name);          // marte
  print(Planeta.marte.distanciaUA);   // 1.52

  for (var p in Planeta.values) {
    print('${p.name}: ${p.distanciaUA} UA');
  }
}
Planeta.values retorna uma lista com todos os valores do enum — útil para percorrer opções sem escrever cada uma manualmente.
Enums no Flutter
Você já usará enums sem perceber no Flutter. Sempre que ver MainAxisAlignment.center, TextAlign.right ou CrossAxisAlignment.start, está usando um enum. Saber como eles funcionam facilita muito a leitura do código Flutter.
Flutter (prévia)
// Todos esses são enums do próprio Flutter:
MainAxisAlignment.center    // alinhamento horizontal
CrossAxisAlignment.start    // alinhamento vertical
TextAlign.right             // alinhamento de texto
Icons.home                  // ícones (também enum)
ThemeMode.dark              // tema escuro/claro
Exercícios de fixação
01
Semáforo
enum + switch básico
Básico
Crie um enum Semaforo com os valores vermelho, amarelo e verde. Declare uma variável Semaforo sinal = Semaforo.amarelo e use switch para exibir: vermelho → "Pare!", amarelo → "Atenção!", verde → "Siga!".
Solução
enum Semaforo { vermelho, amarelo, verde }

void main() {
  Semaforo sinal = Semaforo.amarelo;

  switch (sinal) {
    case Semaforo.vermelho:
      print('Pare!');
    case Semaforo.amarelo:
      print('Atenção!');
    case Semaforo.verde:
      print('Siga!');
  }
}
02
Todos os valores
enum.values + for-in
Básico
Crie um enum Nivel com os valores iniciante, intermediario e avancado. Use Nivel.values e um laço for-in para exibir o .name de cada valor.
Solução
enum Nivel { iniciante, intermediario, avancado }

void main() {
  for (var n in Nivel.values) {
    print(n.name);
  }
}
03
Moeda com valor
enhanced enum com propriedade
Desafio
Crie um enum Moeda com os valores centavo (0.01), real (1.0) e nota10 (10.0), cada um com uma propriedade final double valor. No main(), percorra Moeda.values e exiba o nome e o valor de cada moeda. Depois calcule e exiba o total de todas as moedas somadas.
Solução
enum Moeda {
  centavo(0.01),
  real(1.0),
  nota10(10.0);

  const Moeda(this.valor);
  final double valor;
}

void main() {
  double total = 0;

  for (var m in Moeda.values) {
    print('${m.name}: R\$ ${m.valor}');
    total += m.valor;
  }

  print('Total: R\$ $total');
}
