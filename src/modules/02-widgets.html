Tudo é Widget
Em Flutter, cada elemento visual é um widget — do texto mais simples à tela inteira.
O conceito de Widget
Em Flutter, tudo é widget. Textos, botões, imagens, espaçamentos, cores — cada peça da interface é um widget. Eles se combinam como peças de Lego para formar a tela completa.
StatelessWidget
Sem estado — não muda após renderizado
StatefulWidget
Com estado — pode se atualizar dinamicamente
StatelessWidget
Um StatelessWidget é um widget estático. Você sobrescreve o método build() para dizer o que ele deve desenhar na tela.
Dart
class MeuWidget extends StatelessWidget {
  const MeuWidget({super.key});

  @override
  Widget build(BuildContext context) {
    return const Text('Olá, Flutter!');
  }
}
Widgets essenciais
Dart — Text
Text('Olá Mundo!')

Text(

Dart — Icon e Image
Icon(Icons.star, color: Colors.amber, size: 48)

Image.network('https://exemplo.com/foto.jpg')

Image.asset('assets/imagem.png')
Dart — Container
Container(
  width: 200,
  height: 100,
  color: Colors.blue,
  child: const Text('Dentro do container'),
)
child vs children A propriedade child recebe um único widget filho. Para múltiplos filhos, use children (uma lista) com widgets como Column ou Row.
Exercícios de fixação
01
Primeiro Widget
Criando um StatelessWidget
Crie um StatelessWidget chamado Saudacao que retorna um Text com a mensagem "Bem-vindo ao Flutter!".
Solução
class Saudacao extends StatelessWidget {
  const Saudacao({super.key});

  @override
  Widget build(BuildContext context) {
    return const Text('Bem-vindo ao Flutter!');
  }
}
02
Container colorido
Usando Container com cor e filho
Crie um Container de 150×150 pixels com color: Colors.teal e um filho Center contendo um Text com "Teal".
Solução
Container(
  width: 150,
  height: 150,
  color: Colors.teal,
  child: const Center(
    child: Text('Teal'),
  ),
)
