Módulo 21
Async e Future
Operações demoradas — buscar dados de uma API, ler um arquivo — não podem travar a tela. O Dart resolve isso com Future, async e await.
Future, async e await
Um Future<T> representa um valor que estará disponível no futuro. Marque a função com async e use await para esperar o resultado sem bloquear o restante do app.
Dart
Future<String> buscarNome() async {
  // simula um atraso de rede
  await Future.delayed(const Duration(seconds: 2));
  return 'Tainá';
}

void main() async {
  print('Buscando...');
  final nome = await buscarNome(); // espera 2 segundos
  print('Nome: $nome');             // Nome: Tainá
}
await só pode ser usado dentro de funções marcadas com async. A função sempre retorna um Future — mesmo que você escreva Future<void>, o Dart garante que o retorno seja assíncrono.
FutureBuilder — conectando async à UI
O FutureBuilder é um widget que reconstrói a UI conforme o estado do Future: aguardando, concluído com dados ou concluído com erro. Ele elimina a necessidade de gerenciar manualmente flags de carregamento com setState().
Dart
Future<String> buscarMensagem() async {
  await Future.delayed(const Duration(seconds: 2));
  return 'Dados carregados!';
}

FutureBuilder<String>(
  future: buscarMensagem(),
  builder: (context, snapshot) {
    if (snapshot.connectionState == ConnectionState.waiting) {
      return const CircularProgressIndicator();
    }
    if (snapshot.hasError) {
      return Text('Erro: ${snapshot.error}');
    }
    return Text(snapshot.data!);
  },
)
Declare o Future fora do build() — nunca chame a função diretamente no parâmetro future do FutureBuilder dentro do build(). Isso recriaria o Future a cada reconstrução, reiniciando o carregamento infinitamente.
Tratamento de erros com try/catch
Operações assíncronas podem falhar — sem internet, servidor fora do ar, dados inválidos. Use try/catch dentro de funções async para capturar exceções e exibir feedback adequado ao usuário.
Dart
Future<String> buscarComFalha() async {
  await Future.delayed(const Duration(seconds: 1));
  throw 'Servidor indisponível'; // simula falha
}

Future<void> carregar() async {
  try {
    final dados = await buscarComFalha();
    print(dados);
  } catch (e) {
    print('Erro capturado: $e'); // Erro capturado: Servidor indisponível
  }
}
Boas práticas: sempre trate erros em chamadas assíncronas. Exiba um estado de erro claro na UI — uma mensagem com botão "Tentar novamente" é muito melhor do que uma tela em branco ou travada.
Exercícios de fixação
01
Carregamento simulado
Future.delayed + FutureBuilder
Básico
Crie uma função buscarCitacao() que aguarda 3 segundos com Future.delayed e retorna uma frase motivacional. Use FutureBuilder para exibir um CircularProgressIndicator enquanto carrega e a citação centralizada na tela após o carregamento.
Solução
import 'package:flutter/material.dart';

void main() => runApp(const MaterialApp(home: TelaCitacao()));

Future<String> buscarCitacao() async {
  await Future.delayed(const Duration(seconds: 3));
  return '"O único jeito de fazer um ótimo trabalho\né amar o que você faz." — Steve Jobs';
}

class TelaCitacao extends StatelessWidget {
  const TelaCitacao({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Citação do dia')),
      body: FutureBuilder<String>(
        future: buscarCitacao(),
        builder: (_, snapshot) {
          if (snapshot.connectionState == ConnectionState.waiting) {
            return const Center(child: CircularProgressIndicator());
          }
          return Center(
            child: Padding(
              padding: const EdgeInsets.all(24),
              child: Text(
                snapshot.data!,
                textAlign: TextAlign.center,
                style: const TextStyle(fontSize: 18),
              ),
            ),
          );
        },
      ),
    );
  }
}
02
Recarregar sob demanda
setState + Future + loading manual
Médio
Crie um app que exibe um número aleatório entre 1 e 100 "buscado do servidor". A função buscarNumero() deve aguardar 2 segundos e retornar o número. Use setState() para controlar uma variável _carregando e alternar entre exibir um CircularProgressIndicator ou o número em fonte grande. Um botão "Novo número" dispara outra busca.
Solução
import 'dart:math';
import 'package:flutter/material.dart';

void main() => runApp(const MaterialApp(home: TelaNumero()));

Future<int> buscarNumero() async {
  await Future.delayed(const Duration(seconds: 2));
  return Random().nextInt(100) + 1;
}

class TelaNumero extends StatefulWidget {
  const TelaNumero({super.key});
  @override
  State<TelaNumero> createState() => _TelaNumeroState();
}

class _TelaNumeroState extends State<TelaNumero> {
  int? _numero;
  bool _carregando = false;

  Future<void> _buscar() async {
    setState(() => _carregando = true);
    final n = await buscarNumero();
    setState(() { _numero = n; _carregando = false; });
  }

  @override
  void initState() { super.initState(); _buscar(); }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Número aleatório')),
      body: Center(
        child: _carregando
            ? const CircularProgressIndicator()
            : Text(
                '$_numero',
                style: const TextStyle(fontSize: 72, fontWeight: FontWeight.bold),
              ),
      ),
      floatingActionButton: FloatingActionButton.extended(
        onPressed: _carregando ? null : _buscar,
        label: const Text('Novo número'),
        icon: const Icon(Icons.refresh),
      ),
    );
  }
}
03
Busca com erro recuperável
try/catch + estados: loading, erro, dados
Desafio
Crie um app que simula a busca de um perfil de usuário. A função buscarPerfil(String nome) aguarda 2 segundos: se o nome for "erro", lança uma exceção; caso contrário, retorna um Map com nome, email e cargo. A UI deve ter três estados distintos: carregando (CircularProgressIndicator), erro (mensagem + botão "Tentar novamente") e dados (card com as informações do perfil). Um TextField permite digitar o nome antes de buscar.
Solução
import 'package:flutter/material.dart';

void main() => runApp(const MaterialApp(home: TelaPerfil()));

Future<Map<String, String>> buscarPerfil(String nome) async {
  await Future.delayed(const Duration(seconds: 2));
  if (nome.toLowerCase() == 'erro') throw 'Usuário não encontrado';
  return {
    'nome':  nome,
    'email': '${nome.toLowerCase()}@empresa.com',
    'cargo': 'Desenvolvedor Flutter',
  };
}

class TelaPerfil extends StatefulWidget {
  const TelaPerfil({super.key});
  @override
  State<TelaPerfil> createState() => _TelaPerfilState();
}

class _TelaPerfilState extends State<TelaPerfil> {
  final _ctrl = TextEditingController();
  bool _carregando = false;
  String? _erro;
  Map<String, String>? _perfil;

  @override
  void dispose() { _ctrl.dispose(); super.dispose(); }

  Future<void> _buscar() async {
    final nome = _ctrl.text.trim();
    if (nome.isEmpty) return;
    setState(() { _carregando = true; _erro = null; _perfil = null; });
    try {
      final dados = await buscarPerfil(nome);
      setState(() { _perfil = dados; _carregando = false; });
    } catch (e) {
      setState(() { _erro = e.toString(); _carregando = false; });
    }
  }

  Widget _conteudo() {
    if (_carregando) return const CircularProgressIndicator();
    if (_erro != null) {
      return Column(mainAxisSize: MainAxisSize.min, children: [
        Text(_erro!, style: const TextStyle(color: Colors.red)),
        const SizedBox(height: 12),
        ElevatedButton(onPressed: _buscar, child: const Text('Tentar novamente')),
      ]);
    }
    if (_perfil != null) {
      return Card(
        child: Padding(
          padding: const EdgeInsets.all(24),
          child: Column(mainAxisSize: MainAxisSize.min, children: [
            Text(_perfil!['nome']!, style: const TextStyle(fontSize: 22, fontWeight: FontWeight.bold)),
            const SizedBox(height: 8),
            Text(_perfil!['email']!),
            Text(_perfil!['cargo']!),
          ]),
        ),
      );
    }
    return const Text('Digite um nome para buscar.');
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Buscar Perfil')),
      body: Padding(
        padding: const EdgeInsets.all(24),
        child: Column(children: [
          Row(children: [
            Expanded(
              child: TextField(
                controller: _ctrl,
                decoration: const InputDecoration(hintText: 'Nome do usuário (tente "erro")'),
                onSubmitted: (_) => _buscar(),
              ),
            ),
            IconButton(icon: const Icon(Icons.search), onPressed: _buscar),
          ]),
          const SizedBox(height: 32),
          _conteudo(),
        ]),
      ),
    );
  }
}
