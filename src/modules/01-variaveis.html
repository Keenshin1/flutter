Módulo 01
Variáveis e print()
O ponto de partida de qualquer programa Dart: exibir dados e guardar informações na memória.
O ponto de entrada: main()
Todo programa Dart começa pela função main(). É o ponto de entrada — quando você executa o programa, o Dart procura essa função e começa a rodar o código dentro dela.
Dart
void main() {
  // seu código vai aqui
}
void significa que a função não retorna nenhum valor. Todo programa Dart precisa de uma função main() para ser executado.
Exibindo dados com print()
A função print() exibe qualquer valor no console. É a forma mais simples de ver o resultado do seu código.
Dart
void main() {
  print('Olá, Dart!');
  print(42);
  print(3.14);
  print(true);
}
Saída no console
Olá, Dart!
42
3.14
true
Variáveis
Uma variável guarda um valor na memória. Em Dart você pode usar var (o tipo é inferido automaticamente) ou declarar o tipo de forma explícita.
Dart — com var
void main() {
  var nome   = 'Tainá';  // Dart infere: String
  var idade  = 28;      // Dart infere: int
  var altura = 1.78; // Dart infere: double
  var ativo  = true; // Dart infere: bool

  print(nome);
  print(idade);
  print(altura);
  print(ativo);
}
Dart — tipo explícito
void main() {
  String cidade = 'Curitiba';
  int    ano    = 2025;
  double pi     = 3.14159;
  bool   aberto = false;

  print(cidade);
  print(ano);
}
Regras para nomes de variáveis
Começam com letra ou _ · sem espaços · sensíveis a maiúsculas (nome ≠ Nome) · use camelCase: nomeCompleto, idadeMaxima.
Exercícios de fixação
01
Olá, Dart!
Primeiro programa
Básico
Escreva um programa com main() que exiba "Olá, Dart!" no console.
Solução
void main() {
  print('Olá, Dart!');
}
02
Múltiplos prints
Diferentes tipos de dados
Básico
Escreva um programa que exiba em linhas separadas: seu nome, sua idade (inteiro) e se você é estudante (true/false).
Solução
void main() {
  print('Tainá');
  print(28);
  print(true);
}
03
Cartão de visitas
Variáveis com tipo explícito
Básico
Declare com tipo explícito as variáveis String nome, int idade e double altura. Depois exiba cada uma com print().
Solução
void main() {
  String nome   = 'Tainá';
  int    idade  = 28;
  double altura = 1.78;

  print(nome);
  print(idade);
  print(altura);
}
04
Perfil completo
var, const e interpolação
Desafio
Declare variáveis com var para nome, cidade, ano de nascimento e altura. Use const para o ano atual (2025). Calcule a idade e exiba uma frase no formato: "[nome], [cidade], [idade] anos, [altura]m".
Solução
void main() {
  var nome    = 'Tainá';
  var cidade  = 'Curitiba';
  var anoNasc = 1997;
  var altura  = 1.68;
  const anoAtual = 2025;

  int idade = anoAtual - anoNasc;
  print('$nome, $cidade, $idade anos, ${altura}m');
}
