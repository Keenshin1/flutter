Módulo 10
Classes
Uma classe é um molde para criar objetos. Ela define quais dados (propriedades) e comportamentos (métodos) esses objetos terão.
Declarando uma classe
Dart
class Pessoa {
  String nome;
  int    idade;

  Pessoa(this.nome, this.idade);

  void apresentar() {
    print('Olá, sou $nome e tenho $idade anos.');
  }
}

void main() {
  var p1 = Pessoa('Tainá', 28);
  var p2 = Pessoa('Bruno', 28);

  p1.apresentar(); // Olá, sou Tainá e tenho 28 anos.
  p2.apresentar(); // Olá, sou Bruno e tenho 28 anos.
  print(p1.nome);    // Tainá
}
this.nome no construtor é um atalho do Dart — significa: "pegue o parâmetro nome e atribua à propriedade this.nome".
Construtor nomeado
Dart permite criar construtores alternativos com nomes, úteis para diferentes formas de inicializar um objeto.
Dart
class Ponto {
  double x;
  double y;

  Ponto(this.x, this.y);

  Ponto.origem() : x = 0, y = 0;

  double distanciaOrigem() {
    return (x * x + y * y);
  }
}

void main() {
  var p  = Ponto(3, 4);
  var p0 = Ponto.origem();

  print(p.distanciaOrigem());  // 25.0
  print(p0.x);               // 0.0
}
toString()
Sobrescrever toString() permite que o print() exiba o objeto de forma legível.
Dart
class Produto {
  String nome;
  double preco;

  Produto(this.nome, this.preco);

  @override
  String toString() => 'Produto($nome, R\$ $preco)';
}

void main() {
  var p = Produto('Café', 12.50);
  print(p); // Produto(Café, R$ 12.5)
}
Exercícios de fixação
01
Classe Carro
Propriedades e métodos
Básico
Crie uma classe Carro com as propriedades String marca, String modelo e int ano. Adicione um método descricao() que exiba "[marca] [modelo] ([ano])". Crie dois carros e chame o método.
Solução
class Carro {
  String marca;
  String modelo;
  int    ano;

  Carro(this.marca, this.modelo, this.ano);

  void descricao() =>
      print('$marca $modelo ($ano)');
}

void main() {
  var c1 = Carro('Toyota', 'Corolla', 2022);
  var c2 = Carro('Honda', 'Civic', 2023);
  c1.descricao();
  c2.descricao();
}
02
Produto com toString()
toString() + lista de objetos
Médio
Crie uma classe Produto com String nome, double preco e int estoque. Sobrescreva toString() para retornar "[nome] — R$ [preco] (estoque: [estoque])". Crie uma List<Produto> com 3 itens e exiba-os usando forEach(print).
Solução
class Produto {
  String nome;
  double preco;
  int    estoque;

  Produto(this.nome, this.preco, this.estoque);

  @override
  String toString() =>
      '$nome — R\$ $preco (estoque: $estoque)';
}

void main() {
  var produtos = [
    Produto('Café',      12.50, 3),
    Produto('Leite',     5.90,  15),
    Produto('Manteiga',  18.00, 0),
  ];
  produtos.forEach(print);
}
03
Conta bancária
Classe com lógica de negócio
Desafio
Crie uma classe ContaBancaria com String titular e double saldo. Implemente os métodos: depositar(double v) (soma ao saldo), sacar(double v) (subtrai do saldo, mas exibe "Saldo insuficiente" se o saldo for insuficiente) e sobrescreva toString(). Simule depósitos, saques válidos e um saque inválido.
Solução
class ContaBancaria {
  String titular;
  double saldo;

  ContaBancaria(this.titular, this.saldo);

  void depositar(double v) => saldo += v;

  void sacar(double v) {
    if (v > saldo) {
      print('Saldo insuficiente');
    } else {
      saldo -= v;
    }
  }

  @override
  String toString() => 'Conta de $titular: R\$ $saldo';
}

void main() {
  var conta = ContaBancaria('Tainá', 1000.0);
  conta.depositar(500);
  conta.sacar(200);
  conta.sacar(2000); // Saldo insuficiente
  print(conta);       // Conta de Tainá: R$ 1300.0
}
