Módulo 18
Listas com ListView
Exibir coleções de dados é uma das tarefas mais comuns em apps. O Flutter oferece o ListView — eficiente, rolável e altamente customizável.
ListView e ListTile
O ListView simples recebe uma lista fixa de filhos em children. Para itens estruturados (ícone + título + subtítulo), use o widget ListTile — ele segue automaticamente as proporções do Material Design.
Dart
import 'package:flutter/material.dart';

void main() => runApp(const MaterialApp(home: MinhaLista()));

class MinhaLista extends StatelessWidget {
  const MinhaLista({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Contatos')),
      body: ListView(
        children: const [
          ListTile(
            leading: Icon(Icons.person),
            title: Text('Ana Silva'),
            subtitle: Text('ana@email.com'),
            trailing: Icon(Icons.chevron_right),
          ),
          Divider(),
          ListTile(
            leading: Icon(Icons.person),
            title: Text('Bruno Costa'),
            subtitle: Text('bruno@email.com'),
            trailing: Icon(Icons.chevron_right),
          ),
        ],
      ),
    );
  }
}
ListView com children cria todos os itens de uma vez — adequado só para listas pequenas e fixas. Para listas longas ou dinâmicas, use ListView.builder.
ListView.builder — listas eficientes
O ListView.builder constrói cada item sob demanda, apenas quando ele entra na área visível. É a escolha certa para qualquer lista de tamanho variável — de 10 a 10.000 itens o desempenho se mantém.
Dart
final frutas = ['Maçã', 'Banana', 'Laranja', 'Manga', 'Uva'];

ListView.builder(
  itemCount: frutas.length,
  itemBuilder: (context, index) {
    return ListTile(
      leading: CircleAvatar(child: Text('${index + 1}')),
      title: Text(frutas[index]),
      onTap: () => print('Selecionou: ${frutas[index]}'),
    );
  },
)
itemCount informa quantos itens existem. itemBuilder é chamado com o index atual e deve retornar o widget daquele item. Omitir itemCount cria uma lista infinita.
Lista dinâmica — adicionar e remover itens
Combinando ListView.builder com setState(), é possível adicionar e remover itens em tempo real. A lista em memória é modificada dentro do setState() e o Flutter reconstrói apenas os itens necessários.
Dart
class _ListaState extends State<MinhaLista> {
  final _itens = <String>['Flutter', 'Dart', 'Firebase'];

  void _adicionar(String item) =>
      setState(() => _itens.add(item));

  void _remover(int index) =>
      setState(() => _itens.removeAt(index));

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: ListView.builder(
        itemCount: _itens.length,
        itemBuilder: (_, i) => ListTile(
          title: Text(_itens[i]),
          trailing: IconButton(
            icon: const Icon(Icons.delete),
            onPressed: () => _remover(i),
          ),
        ),
      ),
      floatingActionButton: FloatingActionButton(
        onPressed: () => _adicionar('Novo item ${_itens.length + 1}'),
        child: const Icon(Icons.add),
      ),
    );
  }
}
Padrão comum: mantenha os dados em uma List no State, use setState() para modificá-la e deixe o ListView.builder refletir o estado atual automaticamente.
Exercícios de fixação
01
Lista de linguagens
ListView.builder estático
Básico
Crie uma lista com pelo menos 6 linguagens de programação usando ListView.builder. Cada item deve ser um ListTile com um CircleAvatar exibindo o número do item (1, 2, 3…) como leading, o nome da linguagem como title e um Icon(Icons.code) como trailing.
Solução
import 'package:flutter/material.dart';

void main() => runApp(const MaterialApp(home: ListaLinguagens()));

class ListaLinguagens extends StatelessWidget {
  const ListaLinguagens({super.key});

  static const _linguagens = [
    'Dart', 'Python', 'JavaScript',
    'Kotlin', 'Swift', 'Rust',
  ];

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Linguagens')),
      body: ListView.builder(
        itemCount: _linguagens.length,
        itemBuilder: (_, i) => ListTile(
          leading: CircleAvatar(child: Text('${i + 1}')),
          title: Text(_linguagens[i]),
          trailing: const Icon(Icons.code),
        ),
      ),
    );
  }
}
02
Lista de tarefas
Adicionar e remover com setState
Médio
Crie um app de lista de tarefas. Um TextField no topo (dentro da AppBar ou abaixo dela) permite digitar uma tarefa. Ao pressionar um botão "Adicionar" (ou o FloatingActionButton), a tarefa entra na lista. Cada item da lista tem um IconButton com Icons.delete que remove aquela tarefa. A lista começa vazia.
Solução
import 'package:flutter/material.dart';

void main() => runApp(const MaterialApp(home: ListaTarefas()));

class ListaTarefas extends StatefulWidget {
  const ListaTarefas({super.key});
  @override
  State<ListaTarefas> createState() => _ListaTarefasState();
}

class _ListaTarefasState extends State<ListaTarefas> {
  final _ctrl    = TextEditingController();
  final _tarefas = <String>[];

  @override
  void dispose() { _ctrl.dispose(); super.dispose(); }

  void _adicionar() {
    final texto = _ctrl.text.trim();
    if (texto.isEmpty) return;
    setState(() => _tarefas.add(texto));
    _ctrl.clear();
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Tarefas')),
      body: Column(children: [
        Padding(
          padding: const EdgeInsets.all(12),
          child: Row(children: [
            Expanded(
              child: TextField(
                controller: _ctrl,
                decoration: const InputDecoration(hintText: 'Nova tarefa…'),
                onSubmitted: (_) => _adicionar(),
              ),
            ),
            IconButton(
              icon: const Icon(Icons.add),
              onPressed: _adicionar,
            ),
          ]),
        ),
        Expanded(
          child: _tarefas.isEmpty
              ? const Center(child: Text('Nenhuma tarefa ainda.'))
              : ListView.builder(
                  itemCount: _tarefas.length,
                  itemBuilder: (_, i) => ListTile(
                    title: Text(_tarefas[i]),
                    trailing: IconButton(
                      icon: const Icon(Icons.delete),
                      onPressed: () => setState(() => _tarefas.removeAt(i)),
                    ),
                  ),
                ),
        ),
      ]),
    );
  }
}
03
Lista de compras com total
Objetos na lista + cálculo em tempo real
Desafio
Crie uma lista de compras onde cada item tem nome e preço. Use uma classe Produto para modelar os dados. O app deve ter dois TextField (nome e preço) e um botão para adicionar. A lista exibe cada produto com seu preço. Na parte inferior, mostre o total atualizado automaticamente. Um IconButton com Icons.delete remove o item e recalcula o total.
Solução
import 'package:flutter/material.dart';

void main() => runApp(const MaterialApp(home: ListaCompras()));

class Produto {
  final String nome;
  final double preco;
  Produto(this.nome, this.preco);
}

class ListaCompras extends StatefulWidget {
  const ListaCompras({super.key});
  @override
  State<ListaCompras> createState() => _ListaComprasState();
}

class _ListaComprasState extends State<ListaCompras> {
  final _nomeCtrl  = TextEditingController();
  final _precoCtrl = TextEditingController();
  final _produtos  = <Produto>[];

  @override
  void dispose() {
    _nomeCtrl.dispose();
    _precoCtrl.dispose();
    super.dispose();
  }

  double get _total =>
      _produtos.fold(0, (soma, p) => soma + p.preco);

  void _adicionar() {
    final nome  = _nomeCtrl.text.trim();
    final preco = double.tryParse(_precoCtrl.text);
    if (nome.isEmpty || preco == null) return;
    setState(() => _produtos.add(Produto(nome, preco)));
    _nomeCtrl.clear();
    _precoCtrl.clear();
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Lista de Compras')),
      body: Column(children: [
        Padding(
          padding: const EdgeInsets.all(12),
          child: Row(children: [
            Expanded(
              flex: 2,
              child: TextField(
                controller: _nomeCtrl,
                decoration: const InputDecoration(labelText: 'Produto'),
              ),
            ),
            const SizedBox(width: 8),
            Expanded(
              child: TextField(
                controller: _precoCtrl,
                keyboardType: TextInputType.number,
                decoration: const InputDecoration(labelText: 'Preço'),
              ),
            ),
            IconButton(
              icon: const Icon(Icons.add),
              onPressed: _adicionar,
            ),
          ]),
        ),
        Expanded(
          child: ListView.builder(
            itemCount: _produtos.length,
            itemBuilder: (_, i) => ListTile(
              title: Text(_produtos[i].nome),
              subtitle: Text('R\$ ${_produtos[i].preco.toStringAsFixed(2)}'),
              trailing: IconButton(
                icon: const Icon(Icons.delete),
                onPressed: () => setState(() => _produtos.removeAt(i)),
              ),
            ),
          ),
        ),
        Container(
          width: double.infinity,
          color: Theme.of(context).colorScheme.surfaceVariant,
          padding: const EdgeInsets.all(16),
          child: Text(
            'Total: R\$ ${_total.toStringAsFixed(2)}',
            style: const TextStyle(fontSize: 18, fontWeight: FontWeight.bold),
          ),
        ),
      ]),
    );
  }
}
