Módulo 09
Funções
Funções organizam o código em blocos reutilizáveis. Em Dart, toda função tem um tipo de retorno, um nome e parâmetros opcionais.
Declarando uma função
Dart
int somar(int a, int b) {
  return a + b;
}

void main() {
  int resultado = somar(3, 4);
  print(resultado); // 7
}
O tipo antes do nome (int) é o tipo de retorno. Use void quando a função não retorna nada.
Função com seta (arrow)
Quando o corpo da função é uma única expressão, você pode usar a sintaxe de seta => para encurtar.
Dart
int    dobro(int n)            => n * 2;
String saudar(String nome)    => 'Olá, $nome!';
bool   ehPar(int n)            => n % 2 == 0;

void main() {
  print(dobro(5));        // 10
  print(saudar('Tainá'));  // Olá, Tainá!
  print(ehPar(8));        // true
}
Parâmetros nomeados
Parâmetros entre chaves {} são nomeados — devem ser passados pelo nome na chamada. Use required para tornar obrigatórios.
Dart
String apresentar({
  required String nome,
  int idade = 0,
}) {
  return '$nome, $idade anos';
}

void main() {
  print(apresentar(nome: 'Tainá', idade: 28));
  print(apresentar(nome: 'Tainá')); // idade usa o padrão 0
}
Exercícios de fixação
01
Calcular área
Função com retorno
Básico
Crie uma função double areaRetangulo(double largura, double altura) que retorne a área. Chame-a com alguns valores e exiba o resultado.
Solução
double areaRetangulo(double largura, double altura) =>
    largura * altura;

void main() {
  print(areaRetangulo(5, 3));   // 15.0
  print(areaRetangulo(10, 2.5)); // 25.0
}
02
Saudação personalizada
Parâmetros nomeados
Básico
Crie uma função saudar com parâmetros nomeados required String nome e String periodo = 'dia'. Retorne "Bom [periodo], [nome]!".
Solução
String saudar({
  required String nome,
  String periodo = 'dia',
}) => 'Bom $periodo, $nome!';

void main() {
  print(saudar(nome: 'Tainá'));
  print(saudar(nome: 'Tainá', periodo: 'tarde'));
}
03
Calculadora com funções arrow
Arrow functions + divisão segura
Desafio
Crie 4 funções arrow: somar, subtrair, multiplicar e dividir, todas recebendo dois double. A função dividir deve retornar 0 quando o divisor for 0 — use o operador ternário. Chame cada uma e exiba os resultados, incluindo uma divisão por zero.
Solução
double somar(double a, double b)        => a + b;
double subtrair(double a, double b)     => a - b;
double multiplicar(double a, double b)  => a * b;
double dividir(double a, double b)       => b == 0 ? 0 : a / b;

void main() {
  print(somar(10, 3));        // 13.0
  print(subtrair(10, 3));     // 7.0
  print(multiplicar(10, 3));  // 30.0
  print(dividir(10, 3));       // 3.333...
  print(dividir(5, 0));        // 0.0
}
