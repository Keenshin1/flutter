Módulo 05
Operadores
Operadores são os símbolos que permitem calcular, comparar e combinar valores em Dart.
Operadores aritméticos
Dart
void main() {
  print(10 + 3);  // 13  adição
  print(10 - 3);  // 7   subtração
  print(10 * 3);  // 30  multiplicação
  print(10 / 3);  // 3.333... divisão real
  print(10 ~/ 3); // 3   divisão inteira
  print(10 % 3);  // 1   resto da divisão
}
~/ é o operador de divisão inteira do Dart — descarta a parte decimal. % retorna o resto.
Operadores de comparação
Você já viu esses operadores no módulo anterior. Aqui eles aparecem como referência completa junto com os demais operadores do Dart.
Operadores de comparação sempre retornam um bool (true ou false).
Dart
void main() {
  print(5 == 5);  // true   igual
  print(5 != 3);  // true   diferente
  print(5 >  3);  // true   maior
  print(5 <  3);  // false  menor
  print(5 >= 5);  // true   maior ou igual
  print(5 <= 4);  // false  menor ou igual
}
Operadores lógicos
Dart
void main() {
  bool a = true;
  bool b = false;

  print(a && b); // false  E (ambos precisam ser true)
  print(a || b); // true   OU (basta um ser true)
  print(!a);     // false  NÃO (inverte o valor)
}
Operadores de atribuição
Dart
void main() {
  int x = 10;
  x += 5;  // x = x + 5  → 15
  x -= 3;  // x = x - 3  → 12
  x *= 2;  // x = x * 2  → 24
  x ~/= 4; // x = x ~/ 4 → 6
  print(x); // 6

  int n = 0;
  n++;      // incrementa: n = 1
  n--;      // decrementa: n = 0
  print(n);
}
Exercícios de fixação
01
Calculadora básica
Usando operadores aritméticos
Básico
Dados int a = 17 e int b = 5, exiba: a soma, a diferença, o produto, a divisão inteira e o resto.
Solução
void main() {
  int a = 17;
  int b = 5;
  print(a + b);
  print(a - b);
  print(a * b);
  print(a ~/ b);
  print(a % b);
}
02
Comparações
Testando operadores booleanos
Básico
Dados int x = 8 e int y = 12, exiba: se x é menor que y, se x é igual a y e se x+5 é maior ou igual a y.
Solução
void main() {
  int x = 8;
  int y = 12;
  print(x < y);
  print(x == y);
  print(x + 5 >= y);
}
03
Calculadora de IMC
Combinando operadores aritméticos e lógicos
Desafio
Dados double peso = 70.0 e double altura = 1.75, calcule o IMC (peso / (altura * altura)). Exiba o IMC. Use if/else if/else para exibir a classificação: abaixo de 18.5 → "Abaixo do peso", entre 18.5 e 25 → "Peso normal", entre 25 e 30 → "Sobrepeso", acima → "Obesidade".
Solução
void main() {
  double peso   = 70.0;
  double altura = 1.75;
  double imc    = peso / (altura * altura);

  print('IMC: $imc');

  String classificacao;
  if (imc < 18.5) {
    classificacao = 'Abaixo do peso';
  } else if (imc < 25) {
    classificacao = 'Peso normal';
  } else if (imc < 30) {
    classificacao = 'Sobrepeso';
  } else {
    classificacao = 'Obesidade';
  }
  print(classificacao);
}

Exercício integrador
Combine variáveis, tipos, strings e operadores num único programa.
R
Mini currículo
Variáveis · Tipos · Strings · Operadores
Revisão
Declare variáveis para: String nome, int idade, double altura e bool estudandoFlutter. Use const para o ano atual. Calcule o ano de nascimento. Converta a altura para String com toString(). Exiba tudo numa frase interpolada: "[nome], nascido em [ano], [altura]m, Flutter: [bool]".
Solução
void main() {
  String nome             = 'Tainá';
  int    idade            = 28;
  double altura           = 1.68;
  bool   estudandoFlutter = true;
  const  anoAtual         = 2025;

  int    anoNasc   = anoAtual - idade;
  String alturaStr = altura.toString();

  print('$nome, nascida em $anoNasc, ${alturaStr}m, Flutter: $estudandoFlutter');
}
