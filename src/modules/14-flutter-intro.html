Módulo 14
Introdução ao Flutter
Flutter é um framework onde tudo é widget: textos, botões, layouts e até o próprio aplicativo são widgets que você combina em uma árvore.
Tudo é widget
No Flutter, a interface inteira é construída aninhando widgets dentro de outros widgets — como uma árvore. Cada widget descreve como deve ser exibido; o Flutter cuida de renderizá-lo na tela.
Flutter
// Árvore de widgets — cada nível é um "filho" do anterior
MaterialApp
  └─ Scaffold
       └─ Column
            ├─ Text('Título')
            ├─ Text('Subtítulo')
            └─ Icon(Icons.star)
Tudo que você já aprendeu sobre classes, herança e classes abstratas em Dart é exatamente o que o Flutter usa por baixo. StatelessWidget é uma classe abstrata com o método abstrato build().
StatelessWidget — seu primeiro widget
Um StatelessWidget é um widget que não muda depois de criado. Você estende a classe e implementa o método build(), que retorna a árvore de widgets que representa a interface.
Flutter
import 'package:flutter/material.dart';

class Saudacao extends StatelessWidget {
  const Saudacao({super.key});

  @override
  Widget build(BuildContext context) {
    return const Text('Olá, Flutter!');
  }
}

void main() {
  runApp(const MaterialApp(
    home: Scaffold(body: Center(child: Saudacao())),
  ));
}
const nos widgets é uma otimização importante: diz ao Flutter que o widget nunca vai mudar, permitindo que ele seja reutilizado sem reconstrução. Use sempre que possível.
MaterialApp e Scaffold
MaterialApp configura o aplicativo (tema, rotas, título). Scaffold fornece a estrutura visual padrão do Material Design: barra superior, corpo, botão flutuante e outros.
Flutter
void main() {
  runApp(const MaterialApp(
    title: 'Meu App',
    home: Scaffold(
      appBar: AppBar(title: Text('Início')),
      body: Center(
        child: Text('Conteúdo aqui'),
      ),
    ),
  ));
}

Widgets de layout: Column, Row, Container
Flutter
// Column: empilha widgets na vertical
Column(
  mainAxisAlignment: MainAxisAlignment.center,
  children: [
    Text('Primeiro'),
    SizedBox(height: 8),   // espaço fixo
    Text('Segundo'),
  ],
)

// Row: distribui widgets na horizontal
Row(
  mainAxisAlignment: MainAxisAlignment.spaceEvenly,
  children: [
    Icon(Icons.home),
    Icon(Icons.search),
    Icon(Icons.person),
  ],
)

// Container: caixa com tamanho, cor e padding
Container(
  padding: const EdgeInsets.all(16),
  decoration: BoxDecoration(
    color: Colors.blue.shade50,
    borderRadius: BorderRadius.circular(12),
  ),
  child: Text('Dentro do container'),
)
SizedBox é a forma idiomática de adicionar espaço entre widgets. Use SizedBox(height: 8) para espaço vertical e SizedBox(width: 8) para horizontal, em vez de padding desnecessário.
Exercícios de fixação
Nos exercícios abaixo, escreva apenas a classe do widget — sem o void main(). O ambiente já envolve automaticamente seu widget em um MaterialApp com Scaffold para renderizar o resultado.
01
Cartão de visita
StatelessWidget · Column · Text
Básico
Crie um StatelessWidget chamado CartaoDeVisita que retorna uma Column com mainAxisSize: MainAxisSize.min contendo: nome em negrito (fontSize 22), cargo em cinza (fontSize 14) e e-mail (fontSize 13, Colors.blue). Use SizedBox para separar os itens.
Solução
class CartaoDeVisita extends StatelessWidget {
  const CartaoDeVisita({super.key});

  @override
  Widget build(BuildContext context) {
    return Column(
      mainAxisSize: MainAxisSize.min,
      children: [
        const Text(
          'Ana Souza',
          style: TextStyle(fontSize: 22, fontWeight: FontWeight.bold),
        ),
        const SizedBox(height: 6),
        const Text(
          'Desenvolvedora Flutter',
          style: TextStyle(fontSize: 14, color: Colors.grey),
        ),
        const SizedBox(height: 4),
        const Text(
          'ana@email.com',
          style: TextStyle(fontSize: 13, color: Colors.blue),
        ),
      ],
    );
  }
}
02
Painel informativo
Container · BoxDecoration · Row · Icon
Médio
Crie o widget PainelInfo que retorna um Container com padding: EdgeInsets.all(20) e BoxDecoration com color: Colors.indigo.shade50 e borderRadius de 16. Dentro, uma Column com: (1) uma Row com Icon(Icons.info, color: Colors.indigo) e um Text em negrito separados por SizedBox(width: 8); (2) SizedBox(height: 12); (3) um Text de descrição com color: Colors.black54.
Solução
class PainelInfo extends StatelessWidget {
  const PainelInfo({super.key});

  @override
  Widget build(BuildContext context) {
    return Container(
      padding: const EdgeInsets.all(20),
      decoration: BoxDecoration(
        color: Colors.indigo.shade50,
        borderRadius: BorderRadius.circular(16),
      ),
      child: Column(
        mainAxisSize: MainAxisSize.min,
        crossAxisAlignment: CrossAxisAlignment.start,
        children: [
          Row(
            children: const [
              Icon(Icons.info, color: Colors.indigo),
              SizedBox(width: 8),
              Text(
                'Dica importante',
                style: TextStyle(fontWeight: FontWeight.bold),
              ),
            ],
          ),
          const SizedBox(height: 12),
          const Text(
            'Widgets são imutáveis por padrão. Use StatefulWidget quando precisar de estado que muda.',
            style: TextStyle(color: Colors.black54),
          ),
        ],
      ),
    );
  }
}
03
Tela de perfil
CircleAvatar · Column · Row · layout completo
Desafio
Crie o widget TelaPerfil com a seguinte estrutura: uma Column principal contendo — (1) CircleAvatar(radius: 40, backgroundColor: Colors.deepPurple, child: Text('AS', style: TextStyle(color: Colors.white, fontSize: 24))); (2) nome em negrito (fontSize 20); (3) cargo em cinza; (4) Divider com espaço acima e abaixo; (5) uma bio com texto; (6) uma Row com MainAxisAlignment.spaceEvenly exibindo 3 estatísticas, cada uma sendo uma Column com um número em negrito e um rótulo em cinza (ex: "42 Posts", "1.2k Seguidores", "180 Seguindo").
Solução
class TelaPerfil extends StatelessWidget {
  const TelaPerfil({super.key});

  @override
  Widget build(BuildContext context) {
    return Padding(
      padding: const EdgeInsets.all(24),
      child: Column(
        mainAxisSize: MainAxisSize.min,
        children: [
          const CircleAvatar(
            radius: 40,
            backgroundColor: Colors.deepPurple,
            child: Text(
              'AS',
              style: TextStyle(color: Colors.white, fontSize: 24),
            ),
          ),
          const SizedBox(height: 12),
          const Text(
            'Ana Souza',
            style: TextStyle(fontSize: 20, fontWeight: FontWeight.bold),
          ),
          const SizedBox(height: 4),
          const Text(
            'Desenvolvedora Flutter',
            style: TextStyle(color: Colors.grey),
          ),
          const Divider(height: 32),
          const Text(
            'Apaixonada por UI bonita e código limpo. Criando apps Flutter desde 2020.',
            textAlign: TextAlign.center,
            style: TextStyle(color: Colors.black54),
          ),
          const SizedBox(height: 24),
          Row(
            mainAxisAlignment: MainAxisAlignment.spaceEvenly,
            children: const [
              _Stat(valor: '42',   rotulo: 'Posts'),
              _Stat(valor: '1.2k', rotulo: 'Seguidores'),
              _Stat(valor: '180',  rotulo: 'Seguindo'),
            ],
          ),
        ],
      ),
    );
  }
}

class _Stat extends StatelessWidget {
  final String valor, rotulo;
  const _Stat({required this.valor, required this.rotulo});

  @override
  Widget build(BuildContext context) {
    return Column(
      children: [
        Text(valor,  style: const TextStyle(fontWeight: FontWeight.bold, fontSize: 18)),
        Text(rotulo, style: const TextStyle(color: Colors.grey, fontSize: 12)),
      ],
    );
  }
}
