Módulo 06
Condicionais
Você já conhece o if/else. Agora veja duas formas mais concisas de expressar decisões: o operador ternário e o switch/case.
Operador ternário
Para condições simples, o operador ternário permite escrever um if/else em uma só linha: condição ? valorSeTrue : valorSeFalse
Dart
void main() {
  int x = 15;
  String msg = x % 2 == 0 ? 'par' : 'ímpar';
  print('$x é $msg'); // 15 é ímpar
}
Use o ternário apenas para condições simples — uma linha. Se precisar de lógica mais complexa, prefira o if/else tradicional para manter a legibilidade.
switch / case
Use switch quando precisar comparar uma mesma variável com vários valores possíveis. É mais legível do que uma longa cadeia de else if nesses casos.
Dart
void main() {
  int dia = 3;

  switch (dia) {
    case 1:
      print('Segunda');
    case 2:
      print('Terça');
    case 3:
      print('Quarta');
    default:
      print('Outro dia');
  }
}
No Dart moderno (versão 3+) não é necessário break nos cases — cada case termina automaticamente.
Exercícios de fixação
01
Dia da semana
Usando switch
Básico
Use switch para exibir o nome do dia da semana dado um int dia de 1 a 7. Para valores fora desse intervalo, exiba "Inválido".
Solução
void main() {
  int dia = 4;
  switch (dia) {
    case 1: print('Segunda');
    case 2: print('Terça');
    case 3: print('Quarta');
    case 4: print('Quinta');
    case 5: print('Sexta');
    case 6: print('Sábado');
    case 7: print('Domingo');
    default: print('Inválido');
  }
}
02
Par ou ímpar com ternário
Operador ternário
Básico
Dado int n = 42, use o operador ternário para armazenar em uma variável String resultado o valor "par" ou "ímpar" e exiba: "42 é par".
Solução
void main() {
  int    n         = 42;
  String resultado = n % 2 == 0 ? 'par' : 'ímpar';
  print('$n é $resultado');
}
03
Classificador de temperatura
if/else + ternário combinados
Desafio
Dado int temp = 22, use if/else if/else para classificar: abaixo de 10 → "Frio", entre 10 e 24 → "Agradável", 25 ou mais → "Quente". Em seguida, use o operador ternário para exibir se você precisa de agasalho (quando temp < 15).
Solução
void main() {
  int temp = 22;

  if (temp < 10) {
    print('Frio');
  } else if (temp <= 24) {
    print('Agradável');
  } else {
    print('Quente');
  }

  String agasalho = temp < 15 ? 'Leve agasalho!' : 'Sem agasalho.';
  print(agasalho);
}
