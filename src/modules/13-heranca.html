Herança
Herança permite que uma classe herde propriedades e métodos de outra, evitando repetição e criando hierarquias de tipos — base do Flutter.
extends e @override
Dart
class Animal {
  String nome;

  Animal(this.nome);

  void falar() {
    print('$nome faz algum som.');
  }
}

class Cachorro extends Animal {
  Cachorro(String nome) : super(nome);

  @override
  void falar() {
    print('$nome faz: Au au!');
  }
}

class Gato extends Animal {
  Gato(String nome) : super(nome);

  @override
  void falar() {
    print('$nome faz: Miau!');
  }
}

void main() {
  var animais = [Cachorro('Rex'), Gato('Luna'), Cachorro('Bob')];
  for (var a in animais) {
    a.falar(); // polimorfismo: cada um usa seu próprio falar()
  }
}
super chama o construtor da classe pai. @override indica que estamos substituindo um método herdado — sem isso o Dart exibe um aviso.
Classes abstratas
Uma classe abstract não pode ser instanciada diretamente — ela serve apenas como contrato que as subclasses devem implementar.
Dart
abstract class Forma {
  double area();      // método abstrato: sem corpo
  double perimetro();
}

class Retangulo extends Forma {
  double l, a;
  Retangulo(this.l, this.a);

  @override double area()      => l * a;
  @override double perimetro() => 2 * (l + a);
}

void main() {
  var r = Retangulo(5, 3);
  print(r.area());       // 15.0
  print(r.perimetro()); // 16.0
}

01
Hierarquia de veículos
extends e @override
Básico
Crie uma classe base Veiculo com String marca e método mover(). Crie as subclasses Carro e Moto, cada uma sobreescrevendo mover() com uma mensagem diferente.
Solução
class Veiculo {
  String marca;
  Veiculo(this.marca);
  void mover() => print('$marca se move.');
}

class Carro extends Veiculo {
  Carro(String marca) : super(marca);
  @override
  void mover() => print('$marca anda sobre 4 rodas.');
}

class Moto extends Veiculo {
  Moto(String marca) : super(marca);
  @override
  void mover() => print('$marca anda sobre 2 rodas.');
}

void main() {
  var frota = [Carro('Toyota'), Moto('Honda')];
  for (var v in frota) v.mover();
}
02
Formas geométricas
Classe abstrata com múltiplas implementações
Médio
Crie uma classe abstract Forma com os métodos abstratos double area() e String nome(). Implemente Circulo(double raio) (área = 3.14159 × raio²) e Retangulo(double largura, double altura). Crie uma List<Forma> com 3 formas mistas e exiba o nome e área de cada uma.
Solução
abstract class Forma {
  double area();
  String nome();
}

class Circulo extends Forma {
  double raio;
  Circulo(this.raio);
  @override double area() => 3.14159 * raio * raio;
  @override String nome() => 'Círculo';
}

class Retangulo extends Forma {
  double largura, altura;
  Retangulo(this.largura, this.altura);
  @override double area() => largura * altura;
  @override String nome() => 'Retângulo';
}

void main() {
  List<Forma> formas = [
    Circulo(5),
    Retangulo(4, 6),
    Circulo(3),
  ];
  for (var f in formas) {
    print('${f.nome()}: área = ${f.area()}');
  }
}
03
Carrinho de compras
Composição: classe que contém lista de objetos
Médio
Crie a classe Produto com String nome e double preco. Crie a classe Carrinho que possui uma List<Produto> internamente com os métodos: adicionar(Produto p), double total() (soma dos preços) e exibir() (mostra cada produto e o total). Instancie um carrinho, adicione 3 produtos e chame exibir().
Solução
class Produto {
  String nome;
  double preco;
  Produto(this.nome, this.preco);
}

class Carrinho {
  List<Produto> _itens = [];

  void adicionar(Produto p) => _itens.add(p);

  double total() {
    double soma = 0;
    for (var p in _itens) soma += p.preco;
    return soma;
  }

  void exibir() {
    for (var p in _itens) {
      print('${p.nome}: R\$ ${p.preco}');
    }
    print('Total: R\$ ${total()}');
  }
}

void main() {
  var carrinho = Carrinho();
  carrinho.adicionar(Produto('Caderno', 25.90));
  carrinho.adicionar(Produto('Caneta',  4.50));
  carrinho.adicionar(Produto('Mochila', 149.00));
  carrinho.exibir();
}
04
Simulando widgets Flutter
Hierarquia abstrata inspirada no Flutter
Desafio
Simule a arquitetura do Flutter: crie abstract class Widget com o método abstrato String build(). Implemente Text(String texto) e Button(String rotulo, String acao). Crie uma classe Column que recebe List<Widget> filhos e implementa build() chamando build() de cada filho, separados por quebra de linha. Demonstre criando uma tela com título e dois botões.
Solução
abstract class Widget {
  String build();
}

class Text extends Widget {
  String texto;
  Text(this.texto);
  @override
  String build() => 'Text("$texto")';
}

class Button extends Widget {
  String rotulo, acao;
  Button(this.rotulo, this.acao);
  @override
  String build() => 'Button("$rotulo" → $acao)';
}

class Column extends Widget {
  List<Widget> filhos;
  Column(this.filhos);
  @override
  String build() => filhos.map((w) => '  ${w.build()}').join('\n');
}

void main() {
  var tela = Column([
    Text('Bem-vindo ao Flutter!'),
    Button('Entrar',    'login()'),
    Button('Cadastrar', 'register()'),
  ]);
  print(tela.build());
}

Sistema de funcionários
Classes · Herança · Listas · Polimorfismo
Revisão
Crie uma classe base Funcionario com String nome, double salarioBase e o método double calcularSalario() que retorna o salário base. Crie Gerente(double bonus) e Vendedor(int vendas, double comissaoPorVenda), ambos sobrescrevendo calcularSalario(). Guarde-os em List<Funcionario>, percorra e exiba o nome e salário total de cada um.
Solução
class Funcionario {
  String nome;
  double salarioBase;
  Funcionario(this.nome, this.salarioBase);
  double calcularSalario() => salarioBase;
}

class Gerente extends Funcionario {
  double bonus;
  Gerente(String nome, double base, this.bonus) : super(nome, base);
  @override
  double calcularSalario() => salarioBase + bonus;
}

class Vendedor extends Funcionario {
  int    vendas;
  double comissaoPorVenda;
  Vendedor(String nome, double base, this.vendas, this.comissaoPorVenda)
      : super(nome, base);
  @override
  double calcularSalario() => salarioBase + (vendas * comissaoPorVenda);
}

void main() {
  List<Funcionario> equipe = [
    Gerente('Tainá',  5000, 1500),
    Vendedor('Bruno', 2000, 10, 150),
    Vendedor('Carla', 2000, 7,  150),
  ];
  for (var f in equipe) {
    print('${f.nome}: R\$ ${f.calcularSalario()}');
  }
}
