Módulo 04
Estilo com TextStyle e BoxDecoration
Deixe seus widgets bonitos com cores, fontes, bordas e gradientes.
TextStyle
O TextStyle estiliza um Text: controla fonte, tamanho, peso, cor, espaçamento entre letras e muito mais.
Dart
Text(
  'Flutter é incrível',
  style: TextStyle(
    fontSize: 24,
    fontWeight: FontWeight.bold,
    color: Colors.deepPurple,
    letterSpacing: 1.5,
    fontStyle: FontStyle.italic,
    decoration: TextDecoration.underline,
  ),
)
Colors
O Flutter tem uma paleta completa acessível via Colors. Você também pode usar Color(0xFFRRGGBB) para hex customizado.
Dart
Colors.blue              // azul padrão
Colors.blue[700]          // tonalidade (100–900)
Colors.transparent
Color(0xFF42A5F5)         // hex ARGB completo
Color.fromRGBO(66, 165, 245, 1.0)  // RGB + opacidade
BoxDecoration
O BoxDecoration estiliza um Container: cor, borda, raio dos cantos, gradiente e sombra. Quando usado, substitui a propriedade color direta do Container.
Dart — Card com sombra
Container(
  width: 200,
  height: 80,
  decoration: BoxDecoration(
    color: Colors.blue,
    borderRadius: BorderRadius.circular(16),
    boxShadow: [
      BoxShadow(
        color: Colors.black26,
        blurRadius: 8,
        offset: Offset(0, 4),
      ),
    ],
  ),
  child: const Center(child: Text('Card')),
)
Dart — Gradiente
Container(
  decoration: BoxDecoration(
    gradient: LinearGradient(
      colors: [Colors.blue, Colors.purple],
      begin: Alignment.topLeft,
      end: Alignment.bottomRight,
    ),
  ),
)
Exercícios de fixação
01
Texto estilizado
Usando TextStyle
Crie um Text com o conteúdo "Flutter" usando TextStyle com fontSize: 32, fontWeight: FontWeight.bold e color: Colors.blue.
Solução
Text(
  'Flutter',
  style: TextStyle(
    fontSize: 32,
    fontWeight: FontWeight.bold,
    color: Colors.blue,
  ),
)
02
Card com borda arredondada
Usando BoxDecoration
Crie um Container de 300×100 com BoxDecoration usando color: Colors.indigo e borderRadius: BorderRadius.circular(20). Coloque um Text "Meu Card" em branco centralizado.
Solução
Container(
  width: 300,
  height: 100,
  decoration: BoxDecoration(
    color: Colors.indigo,
    borderRadius: BorderRadius.circular(20),
  ),
  child: const Center(
    child: Text(
      'Meu Card',
      style: TextStyle(color: Colors.white, fontSize: 18),
    ),
  ),
)
