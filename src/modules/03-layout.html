Módulo 03
Layout com Column e Row
Organize widgets vertical e horizontalmente para construir qualquer tipo de tela.
Scaffold e AppBar
O Scaffold é o esqueleto de uma tela. Ele aceita uma AppBar no topo e um body onde vai o conteúdo principal.
Dart
Scaffold(
  appBar: AppBar(
    title: const Text('Meu App'),
    backgroundColor: Colors.blue,
  ),
  body: const Center(
    child: Text('Conteúdo aqui'),
  ),
)
Column e Row
Column organiza widgets um abaixo do outro (eixo vertical). Row organiza widgets lado a lado (eixo horizontal). Ambos aceitam uma lista de filhos em children.
Dart — Column
Column(
  mainAxisAlignment: MainAxisAlignment.center,
  crossAxisAlignment: CrossAxisAlignment.center,
  children: const [
    Text('Linha 1'),
    Text('Linha 2'),
    Text('Linha 3'),
  ],
)
Dart — Row
Row(
  mainAxisAlignment: MainAxisAlignment.spaceEvenly,
  children: const [
    Icon(Icons.home),
    Icon(Icons.search),
    Icon(Icons.person),
  ],
)
mainAxisAlignment start · end · center · spaceBetween · spaceAround · spaceEvenly
Espaçamento e agrupamento
Dart
SizedBox(height: 16)         // espaço fixo vertical
SizedBox(width: 8)           // espaço fixo horizontal

Padding(                      // espaçamento interno
  padding: EdgeInsets.all(16),
  child: const Text('Com padding'),
)

Center(child: const Text('Centralizado'))

Expanded(child: Container(color: Colors.blue)) // ocupa espaço restante
Exercícios de fixação
01
Tela com AppBar
Scaffold completo com Column
Crie um Scaffold com AppBar de título "Minha Tela" e um body com Column centralizada contendo três Text: "Item 1", "Item 2" e "Item 3".
Solução
Scaffold(
  appBar: AppBar(title: const Text('Minha Tela')),
  body: const Column(
    mainAxisAlignment: MainAxisAlignment.center,
    children: [
      Text('Item 1'),
      Text('Item 2'),
      Text('Item 3'),
    ],
  ),
)
02
Barra de ícones
Row com espaçamento uniforme
Crie uma Row com mainAxisAlignment: MainAxisAlignment.spaceEvenly contendo três ícones: Icons.home, Icons.search e Icons.settings.
Solução
Row(
  mainAxisAlignment: MainAxisAlignment.spaceEvenly,
  children: const [
    Icon(Icons.home),
    Icon(Icons.search),
    Icon(Icons.settings),
  ],
)
