Módulo 19
Formulários e entrada de dados
Coletar informações do usuário é essencial em qualquer app. O Flutter oferece TextField, Form e validação integrada para tornar isso seguro e organizado.
Ciclo de vida de um widget com estado
Todo StatefulWidget passa por fases durante sua existência. Conhecer essas fases é essencial para este módulo, porque o TextEditingController precisa ser criado e descartado nos momentos certos.
Dart — ciclo de vida
class _MeuState extends State<MeuWidget> {

  @override
  void initState() {
    super.initState();
    // ① chamado UMA VEZ quando o widget é inserido na tela
    //   → inicialize controllers, streams, requisições
  }

  @override
  Widget build(BuildContext context) {
    // ② chamado toda vez que setState() é chamado
    //   → reconstrói a interface com os novos dados
    return Container();
  }

  @override
  void dispose() {
    // ③ chamado UMA VEZ quando o widget é removido da tela
    //   → cancele controllers, feche streams, libere recursos
    super.dispose();
  }
}
Nunca esqueça de chamar super.initState() e super.dispose() — o Flutter precisa executar sua própria lógica nesses métodos. Omitir o super causa erros difíceis de rastrear.
TextField e TextEditingController
O TextField é o widget de entrada de texto do Flutter. Para ler o valor digitado, use um TextEditingController — ele fica vinculado ao campo e expõe o texto via .text. Crie o controller como campo da classe e sempre descarte-o no dispose() para liberar memória.
Dart
class _FormState extends State<MeuForm> {
  final _controller = TextEditingController();
  String _resultado = '';

  @override
  void dispose() {
    _controller.dispose(); // libera o controller ao destruir o widget
    super.dispose();
  }

  @override
  Widget build(BuildContext context) {
    return Column(children: [
      TextField(
        controller: _controller,
        decoration: const InputDecoration(labelText: 'Seu nome'),
      ),
      ElevatedButton(
        onPressed: () => setState(() => _resultado = _controller.text),
        child: const Text('Confirmar'),
      ),
      Text('Olá, $_resultado!'),
    ]);
  }
}
InputDecoration controla a aparência do campo: labelText (rótulo flutuante), hintText (placeholder), prefixIcon, suffixIcon e muito mais.
Tipos de teclado e campos especiais
O parâmetro keyboardType abre o teclado adequado para cada dado. Use obscureText: true para campos de senha — o Flutter oculta os caracteres automaticamente.
Dart
TextField(
  keyboardType: TextInputType.emailAddress, // abre teclado com @
  decoration: const InputDecoration(labelText: 'E-mail'),
),

TextField(
  keyboardType: TextInputType.number,       // teclado numérico
  decoration: const InputDecoration(labelText: 'Idade'),
),

TextField(
  obscureText: true,                         // oculta os caracteres
  decoration: const InputDecoration(labelText: 'Senha'),
),
Outros valores úteis de TextInputType: .phone, .url, .multiline. Para multiline, combine com maxLines: null para campo de altura livre.
Form e validação com GlobalKey
Para validar múltiplos campos de uma vez, envolva-os em um widget Form e troque TextField por TextFormField. O parâmetro validator recebe o texto e retorna uma mensagem de erro (ou null se válido). Um GlobalKey<FormState> dá acesso ao método validate().
Dart
class _CadastroState extends State<Cadastro> {
  final _formKey = GlobalKey<FormState>();
  final _nomeCtrl  = TextEditingController();
  final _emailCtrl = TextEditingController();

  void _enviar() {
    if (_formKey.currentState!.validate()) {
      print('Enviado: ${_nomeCtrl.text} / ${_emailCtrl.text}');
    }
  }

  @override
  Widget build(BuildContext context) {
    return Form(
      key: _formKey,
      child: Column(children: [
        TextFormField(
          controller: _nomeCtrl,
          decoration: const InputDecoration(labelText: 'Nome'),
          validator: (v) => (v == null || v.isEmpty) ? 'Campo obrigatório' : null,
        ),
        TextFormField(
          controller: _emailCtrl,
          keyboardType: TextInputType.emailAddress,
          decoration: const InputDecoration(labelText: 'E-mail'),
          validator: (v) => (v != null && v.contains('@')) ? null : 'E-mail inválido',
        ),
        ElevatedButton(onPressed: _enviar, child: const Text('Enviar')),
      ]),
    );
  }
}
Fluxo de validação: validate() chama todos os validator do form de uma vez → exibe as mensagens de erro abaixo de cada campo → retorna true só se todos passarem.
Exercícios de fixação
01
Saudação personalizada
TextField + TextEditingController
Básico
Crie um app com um TextField para o usuário digitar o nome e um botão "Saudar". Ao pressionar o botão, exiba abaixo a mensagem "Olá, [nome]! Bem-vindo ao Flutter." usando setState(). Se o campo estiver vazio, exiba "Digite seu nome primeiro."
Solução
import 'package:flutter/material.dart';

void main() => runApp(const MaterialApp(home: Saudacao()));

class Saudacao extends StatefulWidget {
  const Saudacao({super.key});
  @override
  State<Saudacao> createState() => _SaudacaoState();
}

class _SaudacaoState extends State<Saudacao> {
  final _ctrl = TextEditingController();
  String _mensagem = '';

  @override
  void dispose() { _ctrl.dispose(); super.dispose(); }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Saudação')),
      body: Padding(
        padding: const EdgeInsets.all(24),
        child: Column(children: [
          TextField(
            controller: _ctrl,
            decoration: const InputDecoration(labelText: 'Seu nome'),
          ),
          const SizedBox(height: 16),
          ElevatedButton(
            onPressed: () => setState(() {
              final nome = _ctrl.text.trim();
              _mensagem = nome.isEmpty
                  ? 'Digite seu nome primeiro.'
                  : 'Olá, $nome! Bem-vindo ao Flutter.';
            }),
            child: const Text('Saudar'),
          ),
          const SizedBox(height: 16),
          Text(_mensagem, style: const TextStyle(fontSize: 18)),
        ]),
      ),
    );
  }
}
02
Calculadora de IMC
Múltiplos campos numéricos
Médio
Crie um app com dois TextField (peso em kg e altura em m) usando keyboardType: TextInputType.number. Ao pressionar "Calcular", exiba o IMC com duas casas decimais e a classificação: Abaixo do peso (< 18.5), Normal (18.5–24.9), Sobrepeso (25–29.9) ou Obesidade (≥ 30). Trate o caso de campos vazios ou altura zero.
Solução
import 'package:flutter/material.dart';

void main() => runApp(const MaterialApp(home: CalculadoraIMC()));

class CalculadoraIMC extends StatefulWidget {
  const CalculadoraIMC({super.key});
  @override
  State<CalculadoraIMC> createState() => _IMCState();
}

class _IMCState extends State<CalculadoraIMC> {
  final _pesoCtrl   = TextEditingController();
  final _alturaCtrl = TextEditingController();
  String _resultado = '';

  @override
  void dispose() {
    _pesoCtrl.dispose();
    _alturaCtrl.dispose();
    super.dispose();
  }

  void _calcular() {
    final peso   = double.tryParse(_pesoCtrl.text);
    final altura = double.tryParse(_alturaCtrl.text);

    if (peso == null || altura == null || altura == 0) {
      setState(() => _resultado = 'Preencha os campos corretamente.');
      return;
    }

    final imc = peso / (altura * altura);
    final classe = imc < 18.5
        ? 'Abaixo do peso'
        : imc < 25
            ? 'Normal'
            : imc < 30
                ? 'Sobrepeso'
                : 'Obesidade';

    setState(() => _resultado = 'IMC: ${imc.toStringAsFixed(2)} — $classe');
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Calculadora de IMC')),
      body: Padding(
        padding: const EdgeInsets.all(24),
        child: Column(children: [
          TextField(
            controller: _pesoCtrl,
            keyboardType: TextInputType.number,
            decoration: const InputDecoration(labelText: 'Peso (kg)'),
          ),
          TextField(
            controller: _alturaCtrl,
            keyboardType: TextInputType.number,
            decoration: const InputDecoration(labelText: 'Altura (m)'),
          ),
          const SizedBox(height: 16),
          ElevatedButton(onPressed: _calcular, child: const Text('Calcular')),
          const SizedBox(height: 16),
          Text(_resultado, style: const TextStyle(fontSize: 18)),
        ]),
      ),
    );
  }
}
03
Cadastro com validação
Form, TextFormField e GlobalKey
Desafio
Crie um formulário de cadastro usando Form + GlobalKey<FormState> com três campos TextFormField: Nome (obrigatório, mínimo 3 caracteres), E-mail (obrigatório, deve conter @) e Senha (obrigatória, mínimo 6 caracteres, com obscureText: true). O botão "Cadastrar" só avança (exibe um SnackBar de sucesso) se todos os campos passarem na validação.
Solução
import 'package:flutter/material.dart';

void main() => runApp(const MaterialApp(home: TelaCadastro()));

class TelaCadastro extends StatefulWidget {
  const TelaCadastro({super.key});
  @override
  State<TelaCadastro> createState() => _CadastroState();
}

class _CadastroState extends State<TelaCadastro> {
  final _formKey   = GlobalKey<FormState>();
  final _nomeCtrl  = TextEditingController();
  final _emailCtrl = TextEditingController();
  final _senhaCtrl = TextEditingController();

  @override
  void dispose() {
    _nomeCtrl.dispose();
    _emailCtrl.dispose();
    _senhaCtrl.dispose();
    super.dispose();
  }

  void _cadastrar() {
    if (_formKey.currentState!.validate()) {
      ScaffoldMessenger.of(context).showSnackBar(
        SnackBar(content: Text('Bem-vindo, ${_nomeCtrl.text}!')),
      );
    }
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Cadastro')),
      body: Padding(
        padding: const EdgeInsets.all(24),
        child: Form(
          key: _formKey,
          child: Column(children: [
            TextFormField(
              controller: _nomeCtrl,
              decoration: const InputDecoration(labelText: 'Nome'),
              validator: (v) {
                if (v == null || v.trim().isEmpty) return 'Campo obrigatório';
                if (v.trim().length < 3) return 'Mínimo 3 caracteres';
                return null;
              },
            ),
            const SizedBox(height: 12),
            TextFormField(
              controller: _emailCtrl,
              keyboardType: TextInputType.emailAddress,
              decoration: const InputDecoration(labelText: 'E-mail'),
              validator: (v) => (v != null && v.contains('@'))
                  ? null
                  : 'E-mail inválido',
            ),
            const SizedBox(height: 12),
            TextFormField(
              controller: _senhaCtrl,
              obscureText: true,
              decoration: const InputDecoration(labelText: 'Senha'),
              validator: (v) {
                if (v == null || v.isEmpty) return 'Campo obrigatório';
                if (v.length < 6) return 'Mínimo 6 caracteres';
                return null;
              },
            ),
            const SizedBox(height: 24),
            SizedBox(
              width: double.infinity,
              child: ElevatedButton(
                onPressed: _cadastrar,
                child: const Text('Cadastrar'),
              ),
            ),
          ]),
        ),
      ),
    );
  }
}
