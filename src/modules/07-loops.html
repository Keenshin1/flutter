Módulo 06
Loops
Loops executam um bloco de código repetidamente enquanto uma condição for verdadeira ou para cada item de uma coleção.
Loop for
Dart
void main() {
  for (int i = 1; i <= 5; i++) {
    print('Contagem: $i');
  }
}
O for tem três partes: inicialização (int i = 1) · condição (i <= 5) · incremento (i++). Executa enquanto a condição for true.
for-in
Use for-in para percorrer todos os itens de uma lista sem precisar de índice.
Dart
void main() {
  var frutas = ['maçã', 'banana', 'uva'];

  for (var fruta in frutas) {
    print(fruta);
  }
}
while
O while repete o bloco enquanto a condição for true. Use quando não sabe quantas repetições serão necessárias.
Dart
void main() {
  int n = 1;

  while (n <= 5) {
    print('n = $n');
    n++;
  }
}
break e continue
Dart
void main() {
  for (int i = 1; i <= 10; i++) {
    if (i == 4) continue; // pula o 4
    if (i == 7) break;    // para no 7
    print(i);
  }
  // exibe: 1 2 3 5 6
}
Exercícios de fixação
01
Tabuada
Loop for com cálculo
Básico
Use um for para exibir a tabuada do 7 (de 7×1 até 7×10), no formato "7 x 1 = 7".
Solução
void main() {
  for (int i = 1; i <= 10; i++) {
    print('7 x $i = ${7 * i}');
  }
}
02
Só os ímpares
for com continue
Básico
Use um for de 1 a 20 e exiba apenas os números ímpares usando continue para pular os pares.
Solução
void main() {
  for (int i = 1; i <= 20; i++) {
    if (i % 2 == 0) continue;
    print(i);
  }
}
03
FizzBuzz
Loops + condicionais combinados
Desafio
Clássico de programação: percorra os números de 1 a 30 com um for. Para múltiplos de 3 exiba "Fizz", para múltiplos de 5 exiba "Buzz", para múltiplos de ambos exiba "FizzBuzz", e para os demais exiba o próprio número. Dica: teste o múltiplo de 15 primeiro.
Solução
void main() {
  for (int i = 1; i <= 30; i++) {
    if (i % 15 == 0) {
      print('FizzBuzz');
    } else if (i % 3 == 0) {
      print('Fizz');
    } else if (i % 5 == 0) {
      print('Buzz');
    } else {
      print(i);
    }
  }
}

Exercício integrador
Combine condicionais e loops para resolver um problema mais completo.
R
Validador de senhas
Condicionais · Loops · Contadores
Revisão
Dada a lista ['abc', '12345678', 'Ab!3x9z', 'ok', 'segura123'], percorra com for-in. Para cada senha, use if/else para classificar: comprimento ≥ 8 → "Válida", comprimento ≥ 4 → "Fraca", abaixo → "Inválida". Exiba cada senha com sua classificação e, ao final, exiba quantas são válidas.
Solução
void main() {
  var senhas = ['abc', '12345678', 'Ab!3x9z', 'ok', 'segura123'];
  int validas = 0;

  for (var senha in senhas) {
    String status;
    if (senha.length >= 8) {
      status = 'Válida';
      validas++;
    } else if (senha.length >= 4) {
      status = 'Fraca';
    } else {
      status = 'Inválida';
    }
    print('$senha: $status');
  }
  print('Válidas: $validas');
}
