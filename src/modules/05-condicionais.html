Módulo 05
Condicionais
Condicionais permitem que o programa tome decisões e execute caminhos diferentes dependendo de uma condição.
if / else
Dart
void main() {
  int idade = 20;

  if (idade >= 18) {
    print('Maior de idade');
  } else {
    print('Menor de idade');
  }
}
else if
Dart
void main() {
  double nota = 7.5;

  if (nota >= 9) {
    print('Excelente');
  } else if (nota >= 7) {
    print('Aprovado');
  } else if (nota >= 5) {
    print('Recuperação');
  } else {
    print('Reprovado');
  }
}
Operador ternário
Para condições simples, o operador ternário permite escrever um if/else em uma só linha: condição ? valorSeTrue : valorSeFalse
Dart
void main() {
  int x = 15;
  String msg = x % 2 == 0 ? 'par' : 'ímpar';
  print('$x é $msg'); // 15 é ímpar
}
switch / case
Use switch quando precisar comparar uma mesma variável com vários valores possíveis.
Dart
void main() {
  int dia = 3;

  switch (dia) {
    case 1:
      print('Segunda');
    case 2:
      print('Terça');
    case 3:
      print('Quarta');
    default:
      print('Outro dia');
  }
}
No Dart moderno (versão 3+) não é necessário break nos cases — cada case termina automaticamente.
Exercícios de fixação
01
Classificador de notas
if / else if / else
Dada uma variável double nota, classifique: ≥9 → "A", ≥7 → "B", ≥5 → "C", abaixo → "D".
Solução
void main() {
  double nota = 7.5;
  if (nota >= 9) {
    print('A');
  } else if (nota >= 7) {
    print('B');
  } else if (nota >= 5) {
    print('C');
  } else {
    print('D');
  }
}
02
Dia da semana
Usando switch
Use switch para exibir o nome do dia da semana dado um int dia de 1 a 7. Para valores fora desse intervalo, exiba "Inválido".
Solução
void main() {
  int dia = 4;
  switch (dia) {
    case 1: print('Segunda');
    case 2: print('Terça');
    case 3: print('Quarta');
    case 4: print('Quinta');
    case 5: print('Sexta');
    case 6: print('Sábado');
    case 7: print('Domingo');
    default: print('Inválido');
  }
}
