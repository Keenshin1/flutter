Módulo 01
Introdução ao Flutter
Conheça o framework do Google para criar apps bonitos em múltiplas plataformas com um único código.
O que é Flutter?
Flutter é um SDK de código aberto criado pelo Google que permite criar apps para Android, iOS, Web e Desktop usando um único código-base, escrito na linguagem Dart.
Vantagem principal Escreva uma vez, rode em qualquer lugar — o mesmo código funciona em celular, navegador e computador sem reescrever nada.
Estrutura mínima de um app
Todo app Flutter começa com a função main(), que chama runApp() passando o widget raiz. O Flutter então renderiza esse widget na tela.
Dart
import 'package:flutter/material.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return const MaterialApp(
      home: Scaffold(
        body: Center(
          child: Text('Olá Flutter!'),
        ),
      ),
    );
  }
}
Fluxo de execução main() → runApp() → widget raiz → build() → tela renderizada
MaterialApp e Scaffold
MaterialApp configura o app todo (tema, rotas, título). Scaffold oferece a estrutura visual básica de uma tela: AppBar, body, FloatingActionButton etc.

Módulo 01
Variáveis e print()
O ponto de partida de qualquer programa Dart: exibir dados e guardar informações na memória.
O ponto de entrada: main()
Todo programa Dart começa pela função main(). É o ponto de entrada — quando você executa o programa, o Dart procura essa função e começa a rodar o código dentro dela.
Dart
void main() {
  // seu código vai aqui
}
void significa que a função não retorna nenhum valor. Todo programa Dart precisa de uma função main() para ser executado.
Exibindo dados com print()
A função print() exibe qualquer valor no console. É a forma mais simples de ver o resultado do seu código.
Dart
void main() {
  print('Olá, Dart!');
  print(42);
  print(3.14);
  print(true);
}
Saída no console
Olá, Dart!
42
3.14
true
Variáveis
Uma variável guarda um valor na memória. Em Dart você pode usar var (o tipo é inferido automaticamente) ou declarar o tipo de forma explícita.
Dart — com var
void main() {
  var nome   = 'Ana';  // Dart infere: String
  var idade  = 25;    // Dart infere: int
  var altura = 1.68; // Dart infere: double
  var ativo  = true; // Dart infere: bool

  print(nome);
  print(idade);
  print(altura);
  print(ativo);
}
Dart — tipo explícito
void main() {
  String cidade = 'São Paulo';
  int    ano    = 2025;
  double pi     = 3.14159;
  bool   aberto = false;

  print(cidade);
  print(ano);
}
Regras para nomes de variáveis
Começam com letra ou _ · sem espaços · sensíveis a maiúsculas (nome ≠ Nome) · use camelCase: nomeCompleto, idadeMaxima.

Módulo 02
Tipos de Dados
Dart é fortemente tipado — cada variável tem um tipo que determina quais operações você pode fazer com ela.
Os tipos básicos
int
Números inteiros: 0, 42, -7
double
Números decimais: 3.14, -0.5
String
Texto: 'olá', "mundo"
bool
Verdadeiro ou falso
Dart
void main() {
  int    quantidade = 10;
  double preco      = 29.90;
  String produto    = 'Café';
  bool   disponivel = true;

  print(quantidade);
  print(preco);
  print(produto);
  print(disponivel);
}
final e const
Quando um valor não deve mudar após ser definido, use final ou const. Isso deixa o código mais seguro e legible.
Dart
void main() {
  final nome = 'Ana';     // definido em tempo de execução, não muda
  const pi   = 3.14159; // constante em tempo de compilação

  print(nome);
  print(pi);

  // nome = 'outra'; // ERRO: final não pode ser reatribuído
}
final vs const
final é calculado uma vez em tempo de execução. const deve ser um valor completamente conhecido em tempo de compilação (números literais, strings fixas etc.).
Conversão entre tipos
Dart não converte tipos automaticamente. Use os métodos de conversão quando precisar transformar um tipo em outro.
Dart
void main() {
  // String para número
  int    a = int.parse('42');
  double b = double.parse('3.14');

  // Número para String
  String s = 100.toString();

  // int para double e vice-versa
  double d = 5.toDouble();
  int    i = 3.9.toInt();   // trunca: vira 3

  print('$a $b $s $d $i');
}

Tudo é Widget

O conceito de Widget
Em Flutter, tudo é widget. Textos, botões, imagens, espaçamentos, cores — cada peça da interface é um widget. Eles se combinam como peças de Lego para formar a tela completa.

StatefulWidget

StatelessWidget

Layout com Column e Row
Organize widgets vertical e horizontalmente para construir qualquer tipo de tela.
Scaffold e AppBar
O Scaffold é o esqueleto de uma tela. Ele aceita uma AppBar no topo e um body onde vai o conteúdo principal.
Dart
Scaffold(

    title: const Text('Meu App'),

Column e Row
Column organiza widgets um abaixo do outro (eixo vertical). Row organiza widgets lado a lado (eixo horizontal). Ambos aceitam uma lista de filhos em children.
Dart — Column
Column(
  mainAxisAlignment: MainAxisAlignment.center,
  crossAxisAlignment: CrossAxisAlignment.center,
  children: const [
    Text('Linha 1'),
    Text('Linha 2'),
    Text('Linha 3'),
  ],
)
Dart — Row
Row(
  mainAxisAlignment: MainAxisAlignment.spaceEvenly,
  children: const [
    Icon(Icons.home),
    Icon(Icons.search),
    Icon(Icons.person),
  ],
)
mainAxisAlignment start · end · center · spaceBetween · spaceAround · spaceEvenly

Módulo 03
Trabalhando com Strings
Strings são o tipo mais usado em programas reais. Dart oferece interpolação e métodos poderosos para manipulá-las.
Interpolação de strings
Em vez de concatenar strings com +, Dart permite embutir variáveis e expressões diretamente dentro de uma string usando $variavel ou ${expressão}.
Dart
void main() {
  String nome  = 'Ana';
  int    idade = 25;

  print('Olá, $nome!');                        // Olá, Ana!
  print('Idade: $idade anos');                 // Idade: 25 anos
  print('Daqui a 5 anos: ${idade + 5} anos');  // Daqui a 5 anos: 30 anos
  print('Nome em maiúsculas: ${nome.toUpperCase()}');
}
Use $variavel para variáveis simples e ${expressão} quando precisar calcular algo ou chamar um método dentro da string.
Métodos essenciais
Dart
void main() {
  String s = '  Dart é incrivel!  ';

  print(s.length);               // quantidade de caracteres
  print(s.trim());               // remove espaços das bordas
  print(s.toUpperCase());        // DART É INCRIVEL!
  print(s.toLowerCase());        // dart é incrivel!
  print(s.contains('Dart'));    // true
  print(s.replaceAll('!', '.')); // troca caracteres
}
Dart — split e substring
void main() {
  String frase = 'maçã,banana,uva';

  var frutas = frase.split(','); // divide em lista
  print(frutas);              // [maçã, banana, uva]

  print(frase.substring(0, 5)); // maçã,
  print(frase.startsWith('ma')); // true
  print(frase.endsWith('uva'));  // true
}
Strings multilinha
Dart
void main() {
  String texto = '''
Linha 1
Linha 2
Linha 3
''';
  print(texto);
}

Módulo 04
Operadores
Operadores são os símbolos que permitem calcular, comparar e combinar valores em Dart.
Operadores aritméticos
Dart
void main() {
  print(10 + 3);  // 13  adição
  print(10 - 3);  // 7   subtração
  print(10 * 3);  // 30  multiplicação
  print(10 / 3);  // 3.333... divisão real
  print(10 ~/ 3); // 3   divisão inteira
  print(10 % 3);  // 1   resto da divisão
}
~/ é o operador de divisão inteira do Dart — descarta a parte decimal. % retorna o resto.
Operadores de comparação
Operadores de comparação sempre retornam um bool (true ou false).
Dart
void main() {
  print(5 == 5);  // true   igual
  print(5 != 3);  // true   diferente
  print(5 >  3);  // true   maior
  print(5 <  3);  // false  menor
  print(5 >= 5);  // true   maior ou igual
  print(5 <= 4);  // false  menor ou igual
}
Operadores lógicos
Dart
void main() {
  bool a = true;
  bool b = false;

  print(a && b); // false  E (ambos precisam ser true)
  print(a || b); // true   OU (basta um ser true)
  print(!a);     // false  NÃO (inverte o valor)
}
Operadores de atribuição
Dart
void main() {
  int x = 10;
  x += 5;  // x = x + 5  → 15
  x -= 3;  // x = x - 3  → 12
  x *= 2;  // x = x * 2  → 24
  x ~/= 4; // x = x ~/ 4 → 6
  print(x); // 6

  int n = 0;
  n++;      // incrementa: n = 1
  n--;      // decrementa: n = 0
  print(n);
}

Módulo 05
Interatividade com StatefulWidget
Adicione estado e interação aos seus widgets para criar apps dinâmicos.
StatefulWidget
Um StatefulWidget tem estado — dados que podem mudar ao longo do tempo. Quando o estado muda, o Flutter redesenha o widget automaticamente.
Um StatefulWidget é composto de duas classes: a classe do widget e a classe de estado (State).
Dart — Estrutura
class MeuWidget extends StatefulWidget {
  const MeuWidget({super.key});

  @override
  State<MeuWidget> createState() => _MeuWidgetState();
}

class _MeuWidgetState extends State<MeuWidget> {
  int contador = 0;  // variável de estado

  @override
  Widget build(BuildContext context) {
    return Text('$contador');
  }
}
setState()
Para atualizar a tela quando uma variável muda, envolva a mudança dentro de setState(). Isso avisa ao Flutter que ele precisa redesenhar o widget.
Dart
setState(() {
  contador++;  // qualquer mudança de estado aqui
});
Atenção Mudar uma variável sem setState() não atualiza a tela. O Flutter só redesenha quando você chama setState().
Exemplo completo — Contador
Dart — App contador
import 'package:flutter/material.dart';

void main() => runApp(const MaterialApp(home: Contador()));

class Contador extends StatefulWidget {
  const Contador({super.key});

  @override
  State<Contador> createState() => _ContadorState();
}

class _ContadorState extends State<Contador> {
  int _count = 0;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Contador')),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            const Text('Você clicou:'),
            Text(
              '$_count',
              style: const TextStyle(fontSize: 48, fontWeight: FontWeight.bold),
            ),
          ],
        ),
      ),
      floatingActionButton: FloatingActionButton(
        onPressed: () => setState(() => _count++),
        child: const Icon(Icons.add),
      ),
    );
  }
}
