Módulo 01
Variáveis e print()
O ponto de partida de qualquer programa Dart: exibir dados e guardar informações na memória.
O ponto de entrada: main()
Todo programa Dart começa pela função main(). É o ponto de entrada — quando você executa o programa, o Dart procura essa função e começa a rodar o código dentro dela.
Dart
void main() {
  // seu código vai aqui
}
void significa que a função não retorna nenhum valor. Todo programa Dart precisa de uma função main() para ser executado.
Exibindo dados com print()
A função print() exibe qualquer valor no console. É a forma mais simples de ver o resultado do seu código.
Dart
void main() {
  print('Olá, Dart!');
  print(42);
  print(3.14);
  print(true);
}
Saída no console
Olá, Dart!
42
3.14
true
Variáveis
Uma variável guarda um valor na memória. Em Dart você pode usar var (o tipo é inferido automaticamente) ou declarar o tipo de forma explícita.
Dart — com var
void main() {
  var nome   = 'Ana';  // Dart infere: String
  var idade  = 25;    // Dart infere: int
  var altura = 1.68; // Dart infere: double
  var ativo  = true; // Dart infere: bool

  print(nome);
  print(idade);
  print(altura);
  print(ativo);
}
Dart — tipo explícito
void main() {
  String cidade = 'São Paulo';
  int    ano    = 2025;
  double pi     = 3.14159;
  bool   aberto = false;

  print(cidade);
  print(ano);
}
Regras para nomes de variáveis
Começam com letra ou _ · sem espaços · sensíveis a maiúsculas (nome ≠ Nome) · use camelCase: nomeCompleto, idadeMaxima.

Módulo 02
Tipos de Dados
Dart é fortemente tipado — cada variável tem um tipo que determina quais operações você pode fazer com ela.
Os tipos básicos
int
Números inteiros: 0, 42, -7
double
Números decimais: 3.14, -0.5
String
Texto: 'olá', "mundo"
bool
Verdadeiro ou falso
Dart
void main() {
  int    quantidade = 10;
  double preco      = 29.90;
  String produto    = 'Café';
  bool   disponivel = true;

  print(quantidade);
  print(preco);
  print(produto);
  print(disponivel);
}
final e const
Quando um valor não deve mudar após ser definido, use final ou const. Isso deixa o código mais seguro e legible.
Dart
void main() {
  final nome = 'Ana';     // definido em tempo de execução, não muda
  const pi   = 3.14159; // constante em tempo de compilação

  print(nome);
  print(pi);

  // nome = 'outra'; // ERRO: final não pode ser reatribuído
}
final vs const
final é calculado uma vez em tempo de execução. const deve ser um valor completamente conhecido em tempo de compilação (números literais, strings fixas etc.).
Conversão entre tipos
Dart não converte tipos automaticamente. Use os métodos de conversão quando precisar transformar um tipo em outro.
Dart
void main() {
  // String para número
  int    a = int.parse('42');
  double b = double.parse('3.14');

  // Número para String
  String s = 100.toString();

  // int para double e vice-versa
  double d = 5.toDouble();
  int    i = 3.9.toInt();   // trunca: vira 3

  print('$a $b $s $d $i');
}

Módulo 03
Trabalhando com Strings
Strings são o tipo mais usado em programas reais. Dart oferece interpolação e métodos poderosos para manipulá-las.
Interpolação de strings
Em vez de concatenar strings com +, Dart permite embutir variáveis e expressões diretamente dentro de uma string usando $variavel ou ${expressão}.
Dart
void main() {
  String nome  = 'Ana';
  int    idade = 25;

  print('Olá, $nome!');                        // Olá, Ana!
  print('Idade: $idade anos');                 // Idade: 25 anos
  print('Daqui a 5 anos: ${idade + 5} anos');  // Daqui a 5 anos: 30 anos
  print('Nome em maiúsculas: ${nome.toUpperCase()}');
}
Use $variavel para variáveis simples e ${expressão} quando precisar calcular algo ou chamar um método dentro da string.
Métodos essenciais
Dart
void main() {
  String s = '  Dart é incrivel!  ';

  print(s.length);               // quantidade de caracteres
  print(s.trim());               // remove espaços das bordas
  print(s.toUpperCase());        // DART É INCRIVEL!
  print(s.toLowerCase());        // dart é incrivel!
  print(s.contains('Dart'));    // true
  print(s.replaceAll('!', '.')); // troca caracteres
}
Dart — split e substring
void main() {
  String frase = 'maçã,banana,uva';

  var frutas = frase.split(','); // divide em lista
  print(frutas);              // [maçã, banana, uva]

  print(frase.substring(0, 5)); // maçã,
  print(frase.startsWith('ma')); // true
  print(frase.endsWith('uva'));  // true
}
Strings multilinha
Dart
void main() {
  String texto = '''
Linha 1
Linha 2
Linha 3
''';
  print(texto);
}

Módulo 04
Operadores
Operadores são os símbolos que permitem calcular, comparar e combinar valores em Dart.
Operadores aritméticos
Dart
void main() {
  print(10 + 3);  // 13  adição
  print(10 - 3);  // 7   subtração
  print(10 * 3);  // 30  multiplicação
  print(10 / 3);  // 3.333... divisão real
  print(10 ~/ 3); // 3   divisão inteira
  print(10 % 3);  // 1   resto da divisão
}
~/ é o operador de divisão inteira do Dart — descarta a parte decimal. % retorna o resto.
Operadores de comparação
Operadores de comparação sempre retornam um bool (true ou false).
Dart
void main() {
  print(5 == 5);  // true   igual
  print(5 != 3);  // true   diferente
  print(5 >  3);  // true   maior
  print(5 <  3);  // false  menor
  print(5 >= 5);  // true   maior ou igual
  print(5 <= 4);  // false  menor ou igual
}
Operadores lógicos
Dart
void main() {
  bool a = true;
  bool b = false;

  print(a && b); // false  E (ambos precisam ser true)
  print(a || b); // true   OU (basta um ser true)
  print(!a);     // false  NÃO (inverte o valor)
}
Operadores de atribuição
Dart
void main() {
  int x = 10;
  x += 5;  // x = x + 5  → 15
  x -= 3;  // x = x - 3  → 12
  x *= 2;  // x = x * 2  → 24
  x ~/= 4; // x = x ~/ 4 → 6
  print(x); // 6

  int n = 0;
  n++;      // incrementa: n = 1
  n--;      // decrementa: n = 0
  print(n);
}

Módulo 05
Condicionais
Condicionais permitem que o programa tome decisões e execute caminhos diferentes dependendo de uma condição.
if / else
Dart
void main() {
  int idade = 20;

  if (idade >= 18) {
    print('Maior de idade');
  } else {
    print('Menor de idade');
  }
}
else if
Dart
void main() {
  double nota = 7.5;

  if (nota >= 9) {
    print('Excelente');
  } else if (nota >= 7) {
    print('Aprovado');
  } else if (nota >= 5) {
    print('Recuperação');
  } else {
    print('Reprovado');
  }
}
Operador ternário
Para condições simples, o operador ternário permite escrever um if/else em uma só linha: condição ? valorSeTrue : valorSeFalse
Dart
void main() {
  int x = 15;
  String msg = x % 2 == 0 ? 'par' : 'ímpar';
  print('$x é $msg'); // 15 é ímpar
}
switch / case
Use switch quando precisar comparar uma mesma variável com vários valores possíveis.
Dart
void main() {
  int dia = 3;

  switch (dia) {
    case 1:
      print('Segunda');
    case 2:
      print('Terça');
    case 3:
      print('Quarta');
    default:
      print('Outro dia');
  }
}
No Dart moderno (versão 3+) não é necessário break nos cases — cada case termina automaticamente.

Módulo 06
Loops
Loops executam um bloco de código repetidamente enquanto uma condição for verdadeira ou para cada item de uma coleção.
Loop for
Dart
void main() {
  for (int i = 1; i <= 5; i++) {
    print('Contagem: $i');
  }
}
O for tem três partes: inicialização (int i = 1) · condição (i <= 5) · incremento (i++). Executa enquanto a condição for true.
for-in
Use for-in para percorrer todos os itens de uma lista sem precisar de índice.
Dart
void main() {
  var frutas = ['maçã', 'banana', 'uva'];

  for (var fruta in frutas) {
    print(fruta);
  }
}
while
O while repete o bloco enquanto a condição for true. Use quando não sabe quantas repetições serão necessárias.
Dart
void main() {
  int n = 1;

  while (n <= 5) {
    print('n = $n');
    n++;
  }
}
break e continue
Dart
void main() {
  for (int i = 1; i <= 10; i++) {
    if (i == 4) continue; // pula o 4
    if (i == 7) break;    // para no 7
    print(i);
  }
  // exibe: 1 2 3 5 6
}

Módulo 07
Funções
Funções organizam o código em blocos reutilizáveis. Em Dart, toda função tem um tipo de retorno, um nome e parâmetros opcionais.
Declarando uma função
Dart
int somar(int a, int b) {
  return a + b;
}

void main() {
  int resultado = somar(3, 4);
  print(resultado); // 7
}
O tipo antes do nome (int) é o tipo de retorno. Use void quando a função não retorna nada.
Função com seta (arrow)
Quando o corpo da função é uma única expressão, você pode usar a sintaxe de seta => para encurtar.
Dart
int    dobro(int n)            => n * 2;
String saudar(String nome)    => 'Olá, $nome!';
bool   ehPar(int n)            => n % 2 == 0;

void main() {
  print(dobro(5));        // 10
  print(saudar('Ana'));  // Olá, Ana!
  print(ehPar(8));        // true
}
Parâmetros nomeados
Parâmetros entre chaves {} são nomeados — devem ser passados pelo nome na chamada. Use required para tornar obrigatórios.
Dart
String apresentar({
  required String nome,
  int idade = 0,
}) {
  return '$nome, $idade anos';
}

void main() {
  print(apresentar(nome: 'Ana', idade: 25));
  print(apresentar(nome: 'João')); // idade usa o padrão 0
}

Módulo 08
Listas
Uma lista é uma coleção ordenada de valores do mesmo tipo. Em Dart é representada por List<T>.
Criando e acessando listas
Dart
void main() {
  List<String> nomes = ['Ana', 'Bruno', 'Carla'];

  print(nomes[0]);        // Ana   (primeiro elemento)
  print(nomes[2]);        // Carla (terceiro elemento)
  print(nomes.length);   // 3
  print(nomes.first);    // Ana
  print(nomes.last);     // Carla
}
O índice começa em 0. O primeiro elemento é lista[0], o segundo é lista[1], e assim por diante.
Modificando listas
Dart
void main() {
  var nums = [10, 20, 30];

  nums.add(40);          // adiciona ao final
  nums.insert(0, 5);    // insere na posição 0
  nums.remove(20);       // remove o valor 20
  nums.removeAt(0);     // remove pelo índice

  print(nums); // [10, 30, 40]
  print(nums.contains(30)); // true
}
Percorrendo e transformando
Dart
void main() {
  var nums = [1, 2, 3, 4, 5];

  // forEach: percorre cada item
  nums.forEach((n) => print(n));

  // map: transforma cada item
  var dobros = nums.map((n) => n * 2).toList();
  print(dobros); // [2, 4, 6, 8, 10]

  // where: filtra itens
  var pares = nums.where((n) => n % 2 == 0).toList();
  print(pares);  // [2, 4]
}

Módulo 09
Mapas
Um mapa armazena pares de chave → valor. Ideal para representar dados estruturados como configurações, dicionários e registros.
Criando e acessando mapas
Dart
void main() {
  Map<String, int> idades = {
    'Ana':   25,
    'Bruno': 30,
    'Carla': 22,
  };

  print(idades['Ana']);    // 25
  print(idades.length);   // 3
  print(idades.keys);     // (Ana, Bruno, Carla)
  print(idades.values);   // (25, 30, 22)
}
Modificando mapas
Dart
void main() {
  var cores = {'céu': 'azul', 'grama': 'verde'};

  cores['sol'] = 'amarelo';  // adiciona nova entrada
  cores['céu'] = 'cinza';   // atualiza valor existente
  cores.remove('grama');    // remove pela chave

  print(cores.containsKey('sol'));   // true
  print(cores.containsKey('grama')); // false
  print(cores);
}
Percorrendo um mapa
Dart
void main() {
  var notas = {'Ana': 9.0, 'Bruno': 7.5, 'Carla': 8.2};

  // forEach com chave e valor
  notas.forEach((nome, nota) {
    print('$nome: $nota');
  });

  // for-in com entries
  for (var entry in notas.entries) {
    print('${entry.key} tirou ${entry.value}');
  }
}

Módulo 10
Classes
Uma classe é um molde para criar objetos. Ela define quais dados (propriedades) e comportamentos (métodos) esses objetos terão.
Declarando uma classe
Dart
class Pessoa {
  String nome;
  int    idade;

  Pessoa(this.nome, this.idade);

  void apresentar() {
    print('Olá, sou $nome e tenho $idade anos.');
  }
}

void main() {
  var p1 = Pessoa('Ana', 25);
  var p2 = Pessoa('Bruno', 30);

  p1.apresentar(); // Olá, sou Ana e tenho 25 anos.
  p2.apresentar(); // Olá, sou Bruno e tenho 30 anos.
  print(p1.nome);    // Ana
}
this.nome no construtor é um atalho do Dart — significa: "pegue o parâmetro nome e atribua à propriedade this.nome".
Construtor nomeado
Dart permite criar construtores alternativos com nomes, úteis para diferentes formas de inicializar um objeto.
Dart
class Ponto {
  double x;
  double y;

  Ponto(this.x, this.y);

  Ponto.origem() : x = 0, y = 0;

  double distanciaOrigem() {
    return (x * x + y * y);
  }
}

void main() {
  var p  = Ponto(3, 4);
  var p0 = Ponto.origem();

  print(p.distanciaOrigem());  // 25.0
  print(p0.x);               // 0.0
}
toString()
Sobrescrever toString() permite que o print() exiba o objeto de forma legible.
Dart
class Produto {
  String nome;
  double preco;

  Produto(this.nome, this.preco);

  @override
  String toString() => 'Produto($nome, R\$ $preco)';
}

void main() {
  var p = Produto('Café', 12.50);
  print(p); // Produto(Café, R$ 12.5)
}

Módulo 11
Herança
Herança permite que uma classe herde propriedades e métodos de outra, evitando repetição e criando hierarquias de tipos — base do Flutter.
extends e @override
Dart
class Animal {
  String nome;

  Animal(this.nome);

  void falar() {
    print('$nome faz algum som.');
  }
}

class Cachorro extends Animal {
  Cachorro(String nome) : super(nome);

  @override
  void falar() {
    print('$nome faz: Au au!');
  }
}

class Gato extends Animal {
  Gato(String nome) : super(nome);

  @override
  void falar() {
    print('$nome faz: Miau!');
  }
}

void main() {
  var animais = [Cachorro('Rex'), Gato('Luna'), Cachorro('Bob')];
  for (var a in animais) {
    a.falar(); // polimorfismo: cada um usa seu próprio falar()
  }
}
super chama o construtor da classe pai. @override indica que estamos substituindo um método herdado — sem isso o Dart exibe um aviso.
Classes abstratas
Uma classe abstract não pode ser instanciada diretamente — ela serve apenas como contrato que as subclasses devem implementar.
Dart
abstract class Forma {
  double area();      // método abstrato: sem corpo
  double perimetro();
}

class Retangulo extends Forma {
  double l, a;
  Retangulo(this.l, this.a);

  @override double area()      => l * a;
  @override double perimetro() => 2 * (l + a);
}

void main() {
  var r = Retangulo(5, 3);
  print(r.area());       // 15.0
  print(r.perimetro()); // 16.0
}

Exercícios de fixação
01
Hierarquia de veículos
extends e @override
Crie uma classe base Veiculo com String marca e método mover(). Crie as subclasses Carro e Moto, cada uma sobreescrevendo mover() com uma mensagem diferente.
Solução
class Veiculo {
  String marca;
  Veiculo(this.marca);
  void mover() => print('$marca se move.');
}

class Carro extends Veiculo {
  Carro(String marca) : super(marca);
  @override
  void mover() => print('$marca anda sobre 4 rodas.');
}

class Moto extends Veiculo {
  Moto(String marca) : super(marca);
  @override
  void mover() => print('$marca anda sobre 2 rodas.');
}

void main() {
  var frota = [Carro('Toyota'), Moto('Honda')];
  for (var v in frota) v.mover();
}
