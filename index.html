Módulo 01
Introdução ao Flutter
Conheça o framework do Google para criar apps bonitos em múltiplas plataformas com um único código.
O que é Flutter?
Flutter é um SDK de código aberto criado pelo Google que permite criar apps para Android, iOS, Web e Desktop usando um único código-base, escrito na linguagem Dart.
Vantagem principal Escreva uma vez, rode em qualquer lugar — o mesmo código funciona em celular, navegador e computador sem reescrever nada.
Estrutura mínima de um app
Todo app Flutter começa com a função main(), que chama runApp() passando o widget raiz. O Flutter então renderiza esse widget na tela.
Dart
import 'package:flutter/material.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return const MaterialApp(
      home: Scaffold(
        body: Center(
          child: Text('Olá Flutter!'),
        ),
      ),
    );
  }
}
Fluxo de execução main() → runApp() → widget raiz → build() → tela renderizada
MaterialApp e Scaffold
MaterialApp configura o app todo (tema, rotas, título). Scaffold oferece a estrutura visual básica de uma tela: AppBar, body, FloatingActionButton etc.

Módulo 02
Tudo é Widget
Em Flutter, cada elemento visual é um widget — do texto mais simples à tela inteira.
O conceito de Widget
Em Flutter, tudo é widget. Textos, botões, imagens, espaçamentos, cores — cada peça da interface é um widget. Eles se combinam como peças de Lego para formar a tela completa.
StatelessWidget
Sem estado — não muda após renderizado
StatefulWidget
Com estado — pode se atualizar dinamicamente
StatelessWidget
Um StatelessWidget é um widget estático. Você sobrescreve o método build() para dizer o que ele deve desenhar na tela.
Dart
class MeuWidget extends StatelessWidget {
  const MeuWidget({super.key});

  @override
  Widget build(BuildContext context) {
    return const Text('Olá, Flutter!');
  }
}

Módulo 03
Layout com Column e Row
Organize widgets vertical e horizontalmente para construir qualquer tipo de tela.
Scaffold e AppBar
O Scaffold é o esqueleto de uma tela. Ele aceita uma AppBar no topo e um body onde vai o conteúdo principal.
Dart
Scaffold(
  appBar: AppBar(
    title: const Text('Meu App'),
    backgroundColor: Colors.blue,
  ),
  body: const Center(
    child: Text('Conteúdo aqui'),
  ),
)
Column e Row
Column organiza widgets um abaixo do outro (eixo vertical). Row organiza widgets lado a lado (eixo horizontal). Ambos aceitam uma lista de filhos em children.
Dart — Column
Column(
  mainAxisAlignment: MainAxisAlignment.center,
  crossAxisAlignment: CrossAxisAlignment.center,
  children: const [
    Text('Linha 1'),
    Text('Linha 2'),
    Text('Linha 3'),
  ],
)
Dart — Row
Row(
  mainAxisAlignment: MainAxisAlignment.spaceEvenly,
  children: const [
    Icon(Icons.home),
    Icon(Icons.search),
    Icon(Icons.person),
  ],
)
mainAxisAlignment start · end · center · spaceBetween · spaceAround · spaceEvenly
Espaçamento e agrupamento
Dart
SizedBox(height: 16)         // espaço fixo vertical
SizedBox(width: 8)           // espaço fixo horizontal

Padding(                      // espaçamento interno
  padding: EdgeInsets.all(16),
  child: const Text('Com padding'),
)

Center(child: const Text('Centralizado'))

Expanded(child: Container(color: Colors.blue)) // ocupa espaço restante

Módulo 04
Estilo com TextStyle e BoxDecoration
Deixe seus widgets bonitos com cores, fontes, bordas e gradientes.
TextStyle
O TextStyle estiliza um Text: controla fonte, tamanho, peso, cor, espaçamento entre letras e muito mais.
Dart
Text(
  'Flutter é incrível',
  style: TextStyle(
    fontSize: 24,
    fontWeight: FontWeight.bold,
    color: Colors.deepPurple,
    letterSpacing: 1.5,
    fontStyle: FontStyle.italic,
    decoration: TextDecoration.underline,
  ),
)
Colors
O Flutter tem uma paleta completa acessível via Colors. Você também pode usar Color(0xFFRRGGBB) para hex customizado.
Dart
Colors.blue              // azul padrão
Colors.blue[700]          // tonalidade (100–900)
Colors.transparent
Color(0xFF42A5F5)         // hex ARGB completo
Color.fromRGBO(66, 165, 245, 1.0)  // RGB + opacidade
BoxDecoration
O BoxDecoration estiliza um Container: cor, borda, raio dos cantos, gradiente e sombra. Quando usado, substitui a propriedade color direta do Container.
Dart — Card com sombra
Container(
  width: 200,
  height: 80,
  decoration: BoxDecoration(
    color: Colors.blue,
    borderRadius: BorderRadius.circular(16),
    boxShadow: [
      BoxShadow(
        color: Colors.black26,
        blurRadius: 8,
        offset: Offset(0, 4),
      ),
    ],
  ),
  child: const Center(child: Text('Card')),
)
Dart — Gradiente
Container(
  decoration: BoxDecoration(
    gradient: LinearGradient(
      colors: [Colors.blue, Colors.purple],
      begin: Alignment.topLeft,
      end: Alignment.bottomRight,
    ),
  ),
)

Módulo 05
Interatividade com StatefulWidget
Adicione estado e interação aos seus widgets para criar apps dinâmicos.
StatefulWidget
Um StatefulWidget tem estado — dados que podem mudar ao longo do tempo. Quando o estado muda, o Flutter redesenha o widget automaticamente.
Um StatefulWidget é composto de duas classes: a classe do widget e a classe de estado (State).
Dart — Estrutura
class MeuWidget extends StatefulWidget {
  const MeuWidget({super.key});

  @override
  State<MeuWidget> createState() => _MeuWidgetState();
}

class _MeuWidgetState extends State<MeuWidget> {
  int contador = 0;  // variável de estado

  @override
  Widget build(BuildContext context) {
    return Text('$contador');
  }
}
setState()
Para atualizar a tela quando uma variável muda, envolva a mudança dentro de setState(). Isso avisa ao Flutter que ele precisa redesenhar o widget.
Dart
setState(() {
  contador++;  // qualquer mudança de estado aqui
});
Atenção Mudar uma variável sem setState() não atualiza a tela. O Flutter só redesenha quando você chama setState().
Exemplo completo — Contador
Dart — App contador
import 'package:flutter/material.dart';

void main() => runApp(const MaterialApp(home: Contador()));

class Contador extends StatefulWidget {
  const Contador({super.key});

  @override
  State<Contador> createState() => _ContadorState();
}

class _ContadorState extends State<Contador> {
  int _count = 0;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Contador')),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            const Text('Você clicou:'),
            Text(
              '$_count',
              style: const TextStyle(fontSize: 48, fontWeight: FontWeight.bold),
            ),
          ],
        ),
      ),
      floatingActionButton: FloatingActionButton(
        onPressed: () => setState(() => _count++),
        child: const Icon(Icons.add),
      ),
    );
  }
}

02
Botão que incrementa
setState com ElevatedButton
Expanda o widget Placar: adicione um ElevatedButton com texto "Marcar ponto" que, ao ser pressionado, incrementa pontos usando setState(). Exiba os pontos com fontSize: 32.
Solução
class _PlacarState extends State<Placar> {
  int pontos = 0;

  @override
  Widget build(BuildContext context) {
    return Column(
      mainAxisAlignment: MainAxisAlignment.center,
      children: [
        Text(
          'Pontos: $pontos',
          style: const TextStyle(fontSize: 32),
        ),
        const SizedBox(height: 16),
        ElevatedButton(
          onPressed: () => setState(() => pontos++),
          child: const Text('Marcar ponto'),
        ),
      ],
    );
  }
}
